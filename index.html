Comment utiliser ce cours

📖 Lire

Chaque séance est structurée en sections numérotées. Lisez les explications et les exemples.

💻 Pratiquer

Copiez les blocs de code (bouton COPIER) et testez-les dans VS Code + Live Server.

✏️ Exercices

Chaque séance se termine par des exercices pratiques pour ancrer les notions.

🔍 Inspecter

Ouvrez l'inspecteur (F12) sur n'importe quel site pour voir le HTML en action.

💡
Outils recommandés

Téléchargez VS Code sur code.visualstudio.com et installez l'extension Live Server pour voir vos modifications en temps réel. Un navigateur moderne (Chrome ou Firefox) suffira pour tout le reste.
