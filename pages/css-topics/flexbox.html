01

Introduction à Flexbox

Flexbox (Flexible Box Layout) est un modèle de mise en page unidimensionnel — il gère soit une ligne, soit une colonne, mais pas les deux simultanément. Avant l'introduction de Flexbox, l'alignement vertical d'éléments ou la création de colonnes de hauteur égale nécessitaient des techniques CSS complexes et souvent détournées. Flexbox est supporté par l'ensemble des navigateurs modernes depuis 2015. Les problèmes résolus incluent : centrage vertical parfait, colonnes de hauteur égale, espacement uniforme, réordonnancement visuel, et adaptation dynamique à la taille du contenu. Pour une mise en page bidimensionnelle (lignes ET colonnes), CSS Grid est recommandé.

🎮
Analogie : les aimants sur une plaque

Imaginez des aimants disposés sur une plaque métallique. Vous pouvez les organiser en ligne ou en colonne, les espacer, les centrer, les agrandir ou les rétrécir. Flexbox constitue la plaque : vous définissez la direction et l'alignement, et les éléments aimantés suivent automatiquement ces règles.

✅ Problèmes résolus

Centrage vertical parfait, colonnes de hauteur égale, espacement uniforme, réordonnancement visuel, adaptation dynamique à la taille du contenu.

📐 Unidimensionnel

Flexbox opère sur un seul axe à la fois (ligne OU colonne). Pour une mise en page bidimensionnelle (lignes ET colonnes), CSS Grid est recommandé.

🌍 Support navigateur

Flexbox est supporté par l'ensemble des navigateurs modernes depuis 2015.

02

display: flex et l'axe principal

En déclarant display: flex sur un conteneur, ses enfants deviennent des éléments flexibles qui s'alignent automatiquement selon les règles définies. Flexbox repose sur deux axes : l'axe principal (main axis) défini par flex-direction, et l'axe secondaire (cross axis) qui lui est perpendiculaire. La propriété justify-content agit sur l'axe principal, tandis que align-items opère sur l'axe secondaire. display: inline-flex crée un conteneur flex de type inline (taille ajustée au contenu).

Référence complète — display pour Flexbox

ValeurDescriptionContexte d'utilisation
display: flexConteneur flex de type bloc. Occupe toute la largeur disponible.Layout principal, sections, barres de navigation, grilles.
display: inline-flexConteneur flex de type inline. Largeur ajustée au contenu.Éléments qui doivent s'aligner horizontalement sans prendre toute la largeur.
CSS — display: flex
.container { display: flex; } /* Les enfants directs deviennent flexibles */ /* S'alignent horizontalement par défaut */ /* S'étirent pour remplir la hauteur */
📌
Les deux axes de Flexbox

Flexbox repose sur deux axes : l'axe principal (main axis) défini par flex-direction, et l'axe secondaire (cross axis) qui lui est perpendiculaire. La propriété justify-content agit sur l'axe principal, tandis que align-items opère sur l'axe secondaire.

03

flex-direction — changer la direction

flex-direction détermine la direction de l'axe principal. C'est la propriété fondamentale qui définit si les éléments s'alignent en ligne (horizontalement) ou en colonne (verticalement). Les valeurs row-reverse et column-reverse inversent l'ordre visuel des éléments. Combinée à flex-wrap, la direction définit le comportement responsive de base.

Référence complète — flex-direction

ValeurDescriptionContexte d'utilisation
rowValeur par défaut. Axe principal horizontal, de gauche à droite.Barres de navigation, listes horizontales, boutons côte à côte.
row-reverseAxe principal horizontal inversé, de droite à gauche.Designs asymétriques, éléments alignés à droite.
columnAxe principal vertical, de haut en bas.Contenu empilé, cartes en colonne, formulaires.
column-reverseAxe principal vertical inversé, de bas en haut.Chats (messages du bas vers le haut), menus verticaux inversés.
CSS — flex-direction
.row { flex-direction: row; } .row-reverse{ flex-direction: row-reverse; } .column { flex-direction: column; } .column-reverse { flex-direction: column-reverse; }
04

flex-wrap & gap

flex-wrap autorise le retour à la ligne des éléments flexibles. Sans cette propriété (valeur par défaut nowrap), les éléments rétrécissent pour tenir sur une seule ligne, ce qui peut produire des contenus illisibles. flex-wrap: wrap est la base du responsive design avec Flexbox : lorsque les éléments n'ont plus assez d'espace, ils passent automatiquement à la ligne suivante. La propriété flex-flow est un raccourci pour flex-direction et flex-wrap.

Référence complète — flex-wrap

ValeurDescriptionContexte d'utilisation
nowrapValeur par défaut. Tous les éléments tiennent sur une ligne. Ils rétrécissent si nécessaire.Barres de navigation, lignes de boutons qui ne doivent pas passer à la ligne.
wrapLes éléments passent à la ligne suivante quand l'espace est insuffisant.Grilles de cartes, galeries responsives, listes d'icônes.
wrap-reverseLes éléments passent à la ligne en sens inverse (lignes de bas en haut).Layouts inversés, designs non standard.
CSS — flex-wrap
.nowrap { flex-wrap: nowrap; } .wrap { flex-wrap: wrap; } .wrap-rev{ flex-wrap: wrap-reverse; } .flex-flow { flex-flow: row wrap; }
05

justify-content — alignement principal

justify-content distribue l'espace disponible le long de l'axe principal. C'est la propriété clé pour espacer, centrer ou répartir les éléments sur la ligne. La combinaison display: flex; justify-content: center; align-items: center; permet de centrer parfaitement tout élément, horizontalement et verticalement, à l'intérieur de son conteneur — une opération notoirement difficile avant Flexbox.

Référence complète — justify-content

ValeurDescriptionContexte d'utilisation
flex-startValeur par défaut. Éléments alignés au début de l'axe principal.Alignement standard à gauche (row) ou en haut (column).
flex-endÉléments alignés à la fin de l'axe principal.Alignement à droite ou en bas.
centerÉléments centrés sur l'axe principal.Centrage horizontal de boutons, navigation centrée.
space-betweenPremier élément au début, dernier à la fin, espace égal entre les autres.Barre de navigation avec logo à gauche et liens à droite.
space-aroundEspace égal autour de chaque élément (les extrémités ont la moitié de l'espace).Galeries avec espacement visuellement équilibré.
space-evenlyEspace strictement égal entre tous les éléments (y compris les extrémités).Distribution mathématiquement parfaite.
CSS — justify-content
.flex-start { justify-content: flex-start; } .flex-end { justify-content: flex-end; } .center { justify-content: center; } .space-between { justify-content: space-between; } .space-around { justify-content: space-around; } .space-evenly { justify-content: space-evenly; }
💡
Centrage parfait

La combinaison display: flex; justify-content: center; align-items: center; permet de centrer parfaitement tout élément, horizontalement et verticalement, à l'intérieur de son conteneur.

06

align-items — alignement secondaire

align-items détermine l'alignement des éléments sur l'axe secondaire (perpendiculaire à l'axe principal). align-content (différent) distribue l'espace entre les lignes lorsqu'il y a plusieurs lignes (nécessite flex-wrap: wrap). align-self permet de surcharger la valeur pour un élément individuel.

Référence complète — align-items, align-content, align-self

PropriétéValeurDescriptionContexte d'utilisation
align-itemsstretchValeur par défaut. Les éléments s'étirent pour remplir la hauteur du conteneur.Colonnes de hauteur égale.
align-itemsflex-startÉléments alignés en haut de l'axe secondaire.Aligner le contenu en haut.
align-itemsflex-endÉléments alignés en bas de l'axe secondaire.Aligner le contenu en bas.
align-itemscenterÉléments centrés sur l'axe secondaire.Centrage vertical.
align-itemsbaselineÉléments alignés sur la ligne de base du texte.Aligner du texte de tailles différentes.
align-contentflex-start, flex-end, center, space-between, space-around, space-evenly, stretchDistribue l'espace entre les lignes (multi-lignes uniquement).Grilles flexibles avec plusieurs rangées.
align-selfauto, flex-start, flex-end, center, baseline, stretchSurcharge align-items pour un élément spécifique.Un élément particulier doit avoir un alignement différent.
CSS — align-items
.stretch { align-items: stretch; } .flex-start{align-items: flex-start; } .flex-end { align-items: flex-end; } .center { align-items: center; } .baseline { align-items: baseline; }
07

flex-grow, flex-shrink, flex-basis

flex-grow régit la capacité d'agrandissement d'un élément, flex-shrink gère son rétrécissement, et flex-basis détermine sa taille initiale avant répartition de l'espace. La propriété raccourcie flex combine les trois : flex: 1 équivaut à flex-grow: 1; flex-shrink: 1; flex-basis: 0 (l'élément peut s'agrandir et se rétrécir librement). flex: auto équivaut à 1 1 auto (taille de base = contenu). flex: none équivaut à 0 0 auto (taille fixe).

Référence complète — flex-grow, flex-shrink, flex-basis

PropriétéValeursDescriptionContexte d'utilisation
flex-grow0 (défaut) à NFacteur d'agrandissement. 0 = l'élément ne s'agrandit pas. 1+ = part de l'espace disponible.Distribuer l'espace restant : sidebar fixe (0), contenu flexible (1).
flex-shrink1 (défaut) à NFacteur de rétrécissement. 0 = l'élément ne rétrécit pas. 1+ = proportion de réduction.Empêcher un élément de rétrécir (0) : icônes, avatars, tags.
flex-basisauto (défaut), length, percentageTaille initiale avant distribution de l'espace. auto = taille du contenu.Définir une largeur de base : sidebar de 250px, colonne de 30%.
flex: 11 1 0Peut s'agrandir et rétrécir. Taille initiale 0 (toute la place disponible).Contenu principal flexible, colonnes égales.
flex: auto1 1 autoPeut s'agrandir et rétrécir. Taille initiale basée sur le contenu.Éléments qui grandissent mais respectent leur contenu.
flex: none0 0 autoTaille fixe basée sur le contenu. Ne s'agrandit ni ne rétrécit.Éléments qui doivent conserver leur taille exacte.
flex: 0 0 250px0 0 250pxTaille fixe de 250px. Ni agrandissement ni rétrécissement.Sidebar, panneau latéral de largeur fixe.
CSS — Flex grow/shrink/basis
.grow-0 { flex-grow: 0; } .grow-1 { flex-grow: 1; } .grow-2 { flex-grow: 2; } .shrink-0 { flex-shrink: 0; } .basis-200 { flex-basis: 200px; } .basis-auto { flex-basis: auto; } .basis-30 { flex-basis: 30%; } .flex-1 { flex: 1; } .flex-auto { flex: auto; } .flex-none { flex: none; } .sidebar { flex: 0 0 250px; } .main-content { flex: 1; }

08

align-self et order

order modifie l'ordre d'affichage visuel des éléments. Tous les éléments ont un order par défaut de 0. Les éléments sont affichés du plus petit au plus grand order. align-self permet de surcharger la valeur de align-items pour un élément spécifique, permettant à un élément d'avoir un alignement différent des autres. Attention : order modifie l'ordre visuel mais pas l'ordre DOM. Les lecteurs d'écran suivent l'ordre du document HTML. Il est déconseillé d'utiliser order pour modifier l'ordre logique de lecture.

Référence complète — order

ValeurDescriptionContexte d'utilisation
0 (défaut)Ordre neutre. L'élément apparaît dans l'ordre du DOM.Comportement standard.
-1, -2Ordre avant les éléments par défaut.Faire passer un élément en premier visuellement.
1, 2...Ordre après les éléments par défaut.Déplacer un élément en fin de ligne.
CSS — align-self & order
.item-1 { order: 3; } .item-2 { order: 1; } .item-3 { order: 2; } .self-start { align-self: flex-start; } .self-end { align-self: flex-end; } .self-center { align-self: center; } .self-stretch { align-self: stretch; }
⚠️
order et l'accessibilité

La propriété order modifie l'ordre visuel mais pas l'ordre DOM. Les lecteurs d'écran suivent l'ordre du document HTML. Il est déconseillé d'utiliser order pour modifier l'ordre logique de lecture.

09

Gap — espacement entre les items

gap introduit un espacement uniforme entre les éléments flexibles, sans appliquer de marge aux extrémités du conteneur. Les propriétés row-gap et column-gap permettent de différencier l'espacement vertical et horizontal. gap résout le problème des marges qui s'additionnent ou se chevauchent entre éléments flexibles, et simplifie considérablement le code par rapport à l'utilisation de marges individuelles.

Référence complète — gap, row-gap, column-gap

PropriétéDescriptionContexte d'utilisation
gap: sizeEspacement identique entre les lignes et les colonnes.Espacement uniforme dans une grille ou flex.
row-gap: sizeEspacement vertical entre les lignes.Espacement vertical différent de l'horizontal.
column-gap: sizeEspacement horizontal entre les colonnes.Espacement horizontal différent du vertical.
CSS — Gap
.gap-small { gap: 4px; } .gap-medium { gap: 1rem; } .gap-large { gap: 2rem; } .gap-custom { row-gap: 16px; column-gap: 24px; }
10

Projet : Jeu de Cartes

Créez un jeu de cartes adaptatif (responsive) avec Flexbox. Les cartes doivent s'afficher en ligne et passer en colonne sur les écrans de petite taille. Utilisez flex-wrap: wrap pour le responsive sans media queries.

CSS — Jeu de cartes
.card-deck { display: flex; flex-wrap: wrap; gap: 1.5rem; justify-content: center; } .card { flex: 1 1 250px; max-width: 300px; background: white; border-radius: 12px; box-shadow: 0 4px 24px rgba(0,0,0,.08); overflow: hidden; transition: transform .3s, box-shadow .3s; } .card:hover { transform: translateY(-4px); box-shadow: 0 8px 40px rgba(0,0,0,.14); } .card-body { display: flex; flex-direction: column; gap: .5rem; padding: 1.25rem; } .card-tag { align-self: flex-start; background: #e8f4f8; color: #2c7a7b; padding: .25rem .75rem; border-radius: 999px; font-size: .75rem; font-weight: 600; }
  1. Créez un conteneur .card-deck avec display: flex; flex-wrap: wrap; gap: 20px; justify-content: center;.
  2. Ajoutez 3 à 6 cartes avec une classe .card ayant flex: 1 1 280px; max-width: 320px;.
  3. Ajoutez un tag avec align-self: flex-start pour qu'il ne s'étire pas.
  4. Utilisez flex-direction: column sur le corps de la carte (.card-body).
  5. Testez le responsive : les cartes passent automatiquement à la ligne.

Bilan & glossaire

Points-clés à retenir

  • Flexbox est unidimensionnel : ligne OU colonne, pas les deux.
  • display: flex sur le parent active Flexbox.
  • justify-content pour l'axe principal, align-items pour l'axe secondaire.
  • flex-wrap: wrap permet le responsive sans media queries.
  • flex est le raccourci de flex-grow flex-shrink flex-basis.
  • gap remplace les marges pour espacer les items.
  • order change l'ordre visuel, pas l'ordre DOM.
Flex container
Élément parent avec display: flex. Ses enfants directs deviennent des éléments flexibles.
Flex item
Enfant direct d'un conteneur flex. Peut être redimensionné, réordonné et aligné automatiquement.
Main axis / Cross axis
Axe principal défini par flex-direction. L'axe secondaire est perpendiculaire.
flex-grow
Capacité d'un item à s'agrandir si de l'espace est disponible. Valeur par défaut : 0.
flex-shrink
Capacité d'un item à rétrécir si l'espace manque. Valeur par défaut : 1.
flex-basis
Taille initiale d'un item avant répartition de l'espace. Par défaut : auto.
order
Modifie l'ordre visuel des items. Valeur par défaut : 0.
justify-content
Distribue l'espace sur l'axe principal.
align-items
Aligne les items sur l'axe secondaire.
align-self
Surcharge align-items pour un item spécifique.
gap / row-gap / column-gap
Espacement entre les items flexibles.
flex-flow
Raccourci pour flex-direction et flex-wrap.
