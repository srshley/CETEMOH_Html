Introduction à CSS Grid

CSS Grid est le premier véritable système de mise en page bidimensionnel du CSS. Alors que Flexbox gère une seule dimension (ligne OU colonne), Grid prend en charge les deux axes simultanément (lignes ET colonnes). Introduit en 2017 et supporté par l'ensemble des navigateurs modernes, il permet de créer des mises en page complexes avec une syntaxe déclarative et prévisible.

📊
Analogie : le tableau Excel

Des lignes et des colonnes forment des cellules. Chaque cellule peut contenir du contenu, et il est possible de fusionner des cellules (comme grid-column: span 2). CSS Grid fonctionne sur le même principe : une grille bidimensionnelle où les éléments sont placés avec précision.

Flexbox vs Grid

  • Flexbox : unidimensionnel — barres de navigation, centrages, rangées de boutons
  • Grid : bidimensionnel — galeries, pages entières, en-tête + sidebar + contenu + footer
  • Règle : si une dimension → Flexbox ; si deux dimensions → Grid
02

display et grid-template

Pour créer une grille, on déclare display: grid sur le conteneur. Les propriétés grid-template-columns et grid-template-rows définissent la structure de la grille. Le shorthand grid-template combine les deux.

Référence complète — display

ValeurDescriptionContexte d'utilisation
gridÉtablit un contexte de grille en bloc (occupe toute la largeur disponible)Utilisé pour les conteneurs principaux (layout global, sections)
inline-gridÉtablit un contexte de grille en ligne (largeur = contenu)Quand la grille doit s'inscrire dans un flux de texte sans passer à la ligne

Référence complète — grid-template-columns / grid-template-rows

Valeur / SyntaxeDescriptionContexte d'utilisation
1fr 1fr 1fr3 colonnes de largeur égale (parts de l'espace restant)Grille équilibrée où chaque colonne reçoit une part égale
200px 1frColonne fixe de 200 px + colonne flexibleSidebar fixe + contenu principal extensible
1fr 2fr 1frColonnes en proportions 25 % / 50 % / 25 %Mise en page symétrique avec colonne centrale deux fois plus large
repeat(3, 1fr)3 colonnes égales (équivalent à 1fr 1fr 1fr)Évite la répétition manuelle quand plusieurs pistes ont la même taille
autoAjuste la taille au contenu (équivaut à min-content)Colonne qui s'adapte exactement à son contenu
min-contentLa piste fait exactement la taille minimale nécessaire au contenuÉviter le retour à la ligne d'un mot long
max-contentLa piste fait exactement la taille nécessaire sans jamais couper le contenuTitres ou textes qui ne doivent jamais être tronqués
minmax(200px, 1fr)Piste entre 200 px et 1 fr de largeColonne qui a une taille minimale mais peut s'étendre
fit-content(300px)Piste qui s'adapte au contenu sans dépasser 300 pxColonne qui ne dépasse jamais une certaine taille

Référence complète — grid-template (shorthand)

SyntaxeDescriptionExemple
grid-template: rows / columnsDéfinit les lignes et les colonnes en une seule déclarationgrid-template: 100px 1fr / 200px 1fr
grid-template: "areas" rows / columnsCombine template-areas avec les tailles de lignes et colonnesgrid-template: "header header" 80px "main sidebar" 1fr / 1fr 250px
CSS — Live editor
.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem; } /* Shorthand complet */ .layout { display: grid; grid-template: "header header" 80px "sidebar main" 1fr "footer footer" 60px / 250px 1fr; height: 100vh; }

Gap et espacement entre les cellules

gap définit l'espacement entre les cellules de la grille (lignes et colonnes). Il remplace avantageusement l'ancienne technique des marges négatives, et fonctionne également avec Flexbox depuis 2021.

Référence complète — gap

PropriétéDescriptionExemple
gapEspacement uniforme entre toutes les cellules (lignes et colonnes)gap: 16px; — 16 px d'écart partout
row-gapEspacement vertical entre les lignes uniquementrow-gap: 24px;
column-gapEspacement horizontal entre les colonnes uniquementcolumn-gap: 16px;
CSS 
.grid { gap: 16px; }
.grid { row-gap: 24px; column-gap: 16px; }

/* Ancienne syntaxe (dépréciée) : grid-gap, grid-row-gap */
04

Placement explicite : grid-column, grid-row, grid-area

Ces propriétés permettent à un élément de s'étendre sur plusieurs colonnes ou plusieurs lignes, à la manière du colspan / rowspan des tableaux HTML.

Référence complète — grid-column / grid-row

SyntaxeDescriptionContexte d'utilisation
grid-column: span NL'élément s'étend sur N colonnesFusion horizontale simple : un titre qui traverse la grille
grid-column: M / NDe la ligne de grille M à la ligne NPlacement précis entre deux lignes de grille
grid-column: M / span NCommence à M et s'étend sur N colonnesCombinaison début + nombre de cellules à fusionner
grid-column: 1 / -1De la première à la dernière ligne (toute la largeur)Header ou footer qui traversent toute la grille
grid-row: span NL'élément s'étend sur N lignesFusion verticale : article principal sur 2 lignes
grid-row: M / NDe la ligne de grille horizontale M à NPlacement vertical précis
grid-area: nomAssigne l'élément à une zone nommée définie dans grid-template-areasMise en page déclarative : chaque élément reçoit le nom de sa zone
grid-area: row-start / col-start / row-end / col-endShorthand pour les 4 valeurs de placementgrid-area: 1 / 1 / 3 / 4 (ligne 1 à 3, colonne 1 à 4)
CSS 
.item-wide  { grid-column: span 2; }
.item-exact { grid-column: 1 / 3; grid-row: 1 / 3; }
.item-full  { grid-column: 1 / -1; }    /* toute la largeur */
.item { grid-column: 1 / span 2; grid-row: 2 / span 3; }
05

grid-template-areas — mise en page déclarative

La méthode la plus intuitive pour définir une mise en page complète : on "dessine" la grille avec des noms de zones. Un . (point) représente une cellule vide. En responsive, on redéfinit grid-template-areas dans une media query.

Référence complète — grid-template-areas

SyntaxeDescriptionExemple
"nom1 nom2 nom3"Une ligne de la grille ; chaque chaîne entre guillemets correspond à une ligne"header header header"
. (point)Cellule vide dans la grille". main ." — espaces vides à gauche et à droite
Nom identiqueFusionne les cellules adjacentes qui portent le même nom"header header" — header occupe 2 colonnes
grid-area: nomAssigne un élément enfant à une zone définie dans le templateheader { grid-area: header; }
CSS 
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main   main"
    "sidebar main   main"
    "footer footer footer";
  grid-template-columns: 250px 1fr 1fr;
  grid-template-rows: 80px 1fr 1fr 60px;
  height: 100vh;
}
header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }
footer { grid-area: footer; }
06

auto-fill, auto-fit et minmax()

Combinés avec repeat(), auto-fit et auto-fill génèrent des grilles qui adaptent automatiquement le nombre de colonnes à l'espace disponible. C'est la technique de référence pour les galeries responsives sans media queries.

Référence complète — auto-fill / auto-fit

ValeurDescriptionDifférence visuelle
auto-fillCrée autant de colonnes que possible et conserve l'espace des colonnes videsLes colonnes sont tracées même si vides ; la largeur totale est distribuée
auto-fitCrée autant de colonnes que possible et réduit à 0 les colonnes videsLes colonnes vides sont effondrées ; les colonnes existantes s'étirent pour remplir l'espace

Référence complète — minmax()

SyntaxeDescriptionContexte d'utilisation
minmax(200px, 1fr)La piste fait au moins 200 px et au plus 1 frColonne qui a une taille minimale mais peut grandir
minmax(100px, auto)Au moins 100 px, s'adapte au contenu ensuiteHauteur de ligne avec minimum garanti
minmax(min-content, max-content)Entre la taille minimale et maximale du contenuS'adapte exactement au contenu sans forcer un retour à la ligne
CSS 
/* La ligne la plus utile du CSS responsive */
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}

/* auto-fill garde l'espace des colonnes vides */
.gallery-fill {
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
}

/* minmax() basique */
.sidebar-layout {
  grid-template-columns: minmax(200px, 300px) 1fr;
}
07

Grille implicite et grid-auto-flow

Quand le nombre d'éléments dépasse les cellules définies, Grid crée automatiquement de nouvelles pistes. Le placement de ces pistes implicites est contrôlé par grid-auto-flow et leur taille par grid-auto-rows / grid-auto-columns.

Référence complète — grid-auto-flow

ValeurDescriptionContexte d'utilisation
rowPlacement par ligne (valeur par défaut) : les éléments remplissent la première ligne, puis passent à la suivanteGrille standard où la lecture se fait de gauche à droite, haut en bas
columnPlacement par colonne : les éléments remplissent la première colonne, puis passent à la suivanteListes verticales ou timelines
denseActive le mode "tetris" : remplit automatiquement les trous laissés par des éléments spanGaleries d'images où l'ordre n'a pas d'importance
row denseCombinaison : placement par ligne avec remplissage automatique des videsMosaïque dense où on maximise l'occupation de l'espace

Référence complète — grid-auto-columns / grid-auto-rows

PropriétéDescriptionExemple
grid-auto-rows: tailleTaille des lignes implicites créées automatiquementgrid-auto-rows: 80px — toute nouvelle ligne fait 80 px
grid-auto-columns: tailleTaille des colonnes implicites créées automatiquementgrid-auto-columns: 200px — toute nouvelle colonne fait 200 px
CSS 
.grid {
  grid-template-columns: 1fr 1fr 1fr;
  grid-auto-rows: 80px;          /* lignes implicites = 80 px */
  grid-auto-flow: row dense;     /* remplit les trous automatiquement */
}

.grid-column-flow {
  grid-auto-flow: column;
  grid-auto-columns: 200px;
}
08

Alignement dans Grid

Grid offre un contrôle d'alignement complet sur deux niveaux : l'alignement des items dans leur cellule et l'alignement de la grille dans son conteneur.

Référence complète — Alignement des items (dans la cellule)

PropriétéValeursDescription
justify-itemsstart, end, center, stretchAlignement horizontal de tous les items dans leur cellule
align-itemsstart, end, center, stretchAlignement vertical de tous les items dans leur cellule
place-itemsalign justify (shorthand)place-items: center → centrage parfait en 2 lignes (avec display:grid)
justify-selfstart, end, center, stretchAlignement horizontal d'un seul item (surcharge justify-items)
align-selfstart, end, center, stretchAlignement vertical d'un seul item (surcharge align-items)
place-selfalign justify (shorthand)place-self: center end → centré verticalement, collé à droite

Référence complète — Alignement du contenu (grille dans le conteneur)

PropriétéValeursDescription
justify-contentstart, end, center, stretch, space-between, space-around, space-evenlyDistribue l'espace excédentaire horizontalement autour des colonnes
align-contentstart, end, center, stretch, space-between, space-around, space-evenlyDistribue l'espace excédentaire verticalement autour des lignes
place-contentalign justify (shorthand)place-content: center → centre toute la grille dans le conteneur
CSS 
/* Centrage parfait en 2 lignes */
.parent { display: grid; place-items: center; }

/* Alignement individuel */
.item { place-self: start end; }

/* Distribution de la grille */
.grid { place-content: space-between center; }

Fonctions : repeat(), minmax(), fit-content()

Grid introduit des fonctions exclusives qui permettent des grilles flexibles et expressives sans recourir à JavaScript ou à des calculs manuels.

Référence complète — repeat()

SyntaxeDescriptionExemple
repeat(N, taille)Répète taille exactement N foisrepeat(3, 1fr) → 1fr 1fr 1fr
repeat(auto-fill, taille)Répète autant que possible, conserve les pistes videsrepeat(auto-fill, minmax(200px, 1fr))
repeat(auto-fit, taille)Répète autant que possible, effondre les pistes videsrepeat(auto-fit, minmax(200px, 1fr))

Référence complète — fit-content()

SyntaxeDescriptionContexte d'utilisation
fit-content(valeur)La piste s'adapte au contenu mais ne dépasse jamais valeurColonne qui peut rétrécir jusqu'au minimum du contenu mais plafonnée à une largeur max
fit-content(300px)Entre min-content et 300 px, selon le contenuBarre latérale qui ne dépasse jamais 300 px
CSS 
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}

.sidebar-layout {
  grid-template-columns: fit-content(300px) 1fr;
}

.fixed-grid {
  grid-template-columns: repeat(4, 1fr);
}

Exercices pratiques

EX 01 📰 Mise en page magazine
  1. Créez exo-grid-magazine.html avec une grille de 3 colonnes 1fr.
  2. Un article principal occupe 2 colonnes et 2 lignes.
  3. 4 articles secondaires (1 × 1) autour.
  4. Un encart "À la une" (1 colonne, 2 lignes).
  5. Footer traverse toute la largeur (grid-column: 1 / -1).
EX 02 🖼️ Galerie responsive automatique
  1. Créez exo-grid-galerie.html avec repeat(auto-fit, minmax(200px, 1fr)).
  2. Ajoutez 10 images avec object-fit: cover.
  3. La 1ère image fait span 2 sur les colonnes.
  4. Utilisez grid-auto-flow: dense.
🔍
Exercice bonus — Holy Grail Layout

Header, footer, sidebar gauche, sidebar droite, contenu principal. Utilisez grid-template-areas. En mobile (< 768 px), repassez tout en une seule colonne. Un classique des entretiens techniques.

Mini-glossaire

Grid
Système de mise en page bidimensionnel (lignes + colonnes).
fr (fraction)
Unité exclusive à Grid. Distribue l'espace restant en parts proportionnelles.
repeat()
Fonction pour répéter des pistes. Ex : repeat(3, 1fr).
grid-column / grid-row
Fusion de cellules. span N = occupe N colonnes/lignes.
grid-template-areas
Définit des zones nommées. Mise en page déclarative.
auto-fit
Crée autant de colonnes que possible, réduit à 0 les colonnes vides.
auto-fill
Crée autant de colonnes que possible, conserve l'espace des colonnes vides.
minmax(min, max)
Fonction définissant une taille minimale et maximale pour une piste.
fit-content()
Piste qui s'adapte au contenu sans dépasser une valeur maximale.
Piste implicite
Ligne ou colonne créée automatiquement quand les éléments dépassent la grille explicite.
grid-auto-flow: dense
Algorithme de placement qui remplit les trous ("mode tetris").
place-items
Shorthand pour align-items + justify-items. Centre un élément dans sa cellule.
place-content
Shorthand pour align-content + justify-content. Aligne la grille dans le conteneur.
