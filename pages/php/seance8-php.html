01

fopen / fclose

🌍 Navigateur → 📨 Formulaire POST
📨 Formulaire POST → 🖥️ PHP ($_POST)
🖥️ PHP ($_POST) → 📄 Réponse HTML

fopen() ouvre un fichier et retourne un pointeur (resource). fclose() ferme ce pointeur. Toujours fermer un fichier après utilisation.

ModeNomComportement
rLectureOuvre en lecture. Pointeur au début. Fichier doit exister.
wÉcritureOuvre en écriture. Efface le contenu existant. Crée le fichier s'il n'existe pas.
aAjoutOuvre en écriture. Pointeur à la fin. Ajoute sans effacer. Crée le fichier s'il n'existe pas.
r+Lecture/ÉcritureLecture et écriture. Pointeur au début.
w+Lecture/ÉcritureLecture et écriture. Efface le contenu. Crée si inexistant.
a+Lecture/AjoutLecture et ajout. Pointeur à la fin. Crée si inexistant.
PHP — Ouvrir et fermer un fichier 
<?php
// Ouvrir en lecture
$f = fopen('fichier.txt', 'r');
if ($f === false) {
    die("Impossible d'ouvrir le fichier");
}
// ... opérations ...
fclose($f);

// Ouvrir en écriture (efface le contenu)
$f = fopen('nouveau.txt', 'w');
fwrite($f, "Contenu");
fclose($f);

// Ouvrir en ajout
$f = fopen('log.txt', 'a');
fwrite($f, "Nouvelle ligne\n");
fclose($f);

// Toujours vérifier si fopen a réussi
$f = @fopen('inexistant.txt', 'r');
if ($f === false) {
    echo "Le fichier n'existe pas.";
} else {
    fclose($f);
}
?>
02

fwrite — écrire dans un fichier

PHP — Écrire dans un fichier 
<?php
// fwrite(ressource, chaîne, longueur_max_optionnelle)
$f = fopen('journal.txt', 'a');

$ligne = date('Y-m-d H:i:s') . " - Action effectuée\n";
fwrite($f, $ligne);

fclose($f);

// Créer un fichier CSV
$f = fopen('utilisateurs.csv', 'w');
fwrite($f, "nom,prenom,email\n");
fwrite($f, "Dupont,Alice,alice@exemple.com\n");
fwrite($f, "Martin,Bob,bob@exemple.com\n");
fclose($f);

// fputs() est un alias de fwrite()
// Alternative moderne : file_put_contents (section 04)
?>

03

fgets / file — lire un fichier

PHP — Lire un fichier 
<?php
// fgets — lit une ligne à la fois
$f = fopen('journal.txt', 'r');
while (($ligne = fgets($f)) !== false) {
    echo htmlspecialchars($ligne) . "<br>";
}
fclose($f);

// file — lit tout le fichier dans un tableau (une ligne par élément)
$lignes = file('journal.txt');
if ($lignes !== false) {
    foreach ($lignes as $num => $ligne) {
        echo "Ligne " . ($num + 1) . " : " . htmlspecialchars($ligne) . "<br>";
    }
}

// fgetcsv — lit un fichier CSV ligne par ligne
$f = fopen('utilisateurs.csv', 'r');
while (($data = fgetcsv($f)) !== false) {
    print_r($data);   // tableau contenant les champs
}
fclose($f);

// fread — lit un nombre d'octets précis
$f = fopen('fichier.txt', 'r');
$contenu = fread($f, 1024);   // lit 1024 octets
fclose($f);
?>

04

file_get_contents / file_put_contents

Ces deux fonctions sont des raccourcis pratiques qui évitent de gérer fopen/fclose. Elles sont idéales pour les opérations simples.

PHP — Raccourcis de lecture/écriture 
<?php
// Lire tout le fichier dans une chaîne
$contenu = file_get_contents('fichier.txt');
echo nl2br(htmlspecialchars($contenu));

// Lire une URL distante
$json = file_get_contents('https://api.exemple.com/data');
$data = json_decode($json, true);

// Écrire tout le fichier d'un coup
$texte = "Nouveau contenu du fichier\nLigne 2\nLigne 3";
file_put_contents('nouveau.txt', $texte);

// Ajouter au lieu d'écraser (FILE_APPEND)
file_put_contents('log.txt', "Nouvelle entrée\n", FILE_APPEND);

// Verrouiller le fichier (LOCK_EX) pour éviter les écritures concurrentes
file_put_contents('compteur.txt', '42', LOCK_EX);

// file_get_contents avec contexte HTTP
$options = [
    'http' => [
        'header' => "User-Agent: MonApp/1.0\r\n",
        'method' => 'GET',
    ]
];
$context = stream_context_create($options);
$resultat = file_get_contents('https://api.exemple.com', false, $context);
?>

06

move_uploaded_file

Le fichier uploadé est stocké temporairement. Pour le conserver, il faut le déplacer avec move_uploaded_file().

PHP — Déplacer un fichier uploadé 
<?php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $dossier = 'uploads/';

    // Créer le dossier s'il n'existe pas
    if (!is_dir($dossier)) {
        mkdir($dossier, 0755, true);
    }

    // Chemin de destination
    $nomFichier = basename($_FILES['photo']['name']);
    $destination = $dossier . $nomFichier;

    // Déplacer le fichier
    if (move_uploaded_file($_FILES['photo']['tmp_name'], $destination)) {
        echo "Fichier uploadé avec succès : $destination";
    } else {
        echo "Erreur lors de l'upload.";
    }
}

// Toujours utiliser basename() pour éviter les attaques de type
// path traversal : ../../../etc/passwd
// Générer un nom unique pour éviter les collisions
$extension = pathinfo($_FILES['photo']['name'], PATHINFO_EXTENSION);
$nouveauNom = uniqid() . '.' . $extension;
$destination = $dossier . $nouveauNom;
?>
⚠️
Sécurité : path traversal

Ne faites jamais confiance au nom de fichier fourni par l'utilisateur. Utilisez basename() et générez un nom unique avec uniqid() ou md5(uniqid()) pour éviter les collisions et les attaques.

07

Validation de l'upload

Valider le type, la taille et les dimensions des fichiers uploadés est essentiel pour la sécurité.

PHP — Validation complète d'un upload 
<?php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $fichier = $_FILES['photo'] ?? null;
    $erreurs = [];

    if (!$fichier || $fichier['error'] !== UPLOAD_ERR_OK) {
        $erreurs[] = "Erreur lors de l'upload.";
    } else {
        // 1. Vérifier la taille (max 5 Mo)
        $maxSize = 5 * 1024 * 1024;  // 5 Mo
        if ($fichier['size'] > $maxSize) {
            $erreurs[] = "Le fichier dépasse la taille maximale de 5 Mo.";
        }

        // 2. Vérifier le type MIME réel (pas celui déclaré par le client)
        $finfo = finfo_open(FILEINFO_MIME_TYPE);
        $mime = finfo_file($finfo, $fichier['tmp_name']);
        finfo_close($finfo);

        $typesAutorises = ['image/jpeg', 'image/png', 'image/gif', 'image/webp'];
        if (!in_array($mime, $typesAutorises)) {
            $erreurs[] = "Type de fichier non autorisé. Formats : JPG, PNG, GIF, WebP.";
        }

        // 3. Vérifier l'extension
        $extension = strtolower(pathinfo($fichier['name'], PATHINFO_EXTENSION));
        $extensions = ['jpg', 'jpeg', 'png', 'gif', 'webp'];
        if (!in_array($extension, $extensions)) {
            $erreurs[] = "Extension non autorisée.";
        }

        // 4. Vérifier les dimensions (pour les images)
        if (in_array($mime, ['image/jpeg', 'image/png', 'image/gif', 'image/webp'])) {
            $dimensions = getimagesize($fichier['tmp_name']);
            if ($dimensions[0] > 4000 || $dimensions[1] > 4000) {
                $erreurs[] = "L'image est trop grande (max 4000×4000 px).";
            }
        }

        // Si tout est OK, déplacer le fichier
        if (empty($erreurs)) {
            $nouveauNom = uniqid('img_') . '.' . $extension;
            $destination = 'uploads/' . $nouveauNom;
            if (move_uploaded_file($fichier['tmp_name'], $destination)) {
                $succes = "Fichier uploadé : " . htmlspecialchars($nouveauNom);
            } else {
                $erreurs[] = "Erreur lors de la sauvegarde.";
            }
        }
    }
}
?>

Gestion des erreurs fichiers

PHP fournit des codes d'erreur standard pour les opérations sur les fichiers. Les voici pour $_FILES['photo']['error'].

CodeConstanteSignification
0UPLOAD_ERR_OKSuccès
1UPLOAD_ERR_INI_SIZEFichier dépasse upload_max_filesize (php.ini)
2UPLOAD_ERR_FORM_SIZEFichier dépasse MAX_FILE_SIZE (formulaire HTML)
3UPLOAD_ERR_PARTIALFichier partiellement reçu
4UPLOAD_ERR_NO_FILEAucun fichier sélectionné
6UPLOAD_ERR_NO_TMP_DIRDossier temporaire manquant
7UPLOAD_ERR_CANT_WRITEImpossible d'écrire sur le disque
8UPLOAD_ERR_EXTENSIONExtension PHP a bloqué l'upload
PHP — Gestion des erreurs systèmes 
<?php
// Gestion des erreurs fopen
$f = @fopen('config.php', 'r');
if ($f === false) {
    $erreur = error_get_last();
    echo "Erreur : " . $erreur['message'];
} else {
    fclose($f);
}

// Vérifier les permissions
$fichier = 'donnees.txt';
if (!file_exists($fichier)) {
    touch($fichier);   // crée le fichier vide
}
if (!is_writable($fichier)) {
    chmod($fichier, 0644);   // tenter de donner les droits
}

// Exceptions avec try/catch (PHP 7+)
try {
    $contenu = file_get_contents('fichier.txt');
    if ($contenu === false) {
        throw new Exception("Impossible de lire le fichier");
    }
} catch (Exception $e) {
    echo "Erreur : " . $e->getMessage();
}
?>

Exercices pratiques

EX 01 ✏️ Journal de bord
  1. Créez journal.php avec un formulaire (texte + bouton "Ajouter").
  2. Chaque soumission ajoute une ligne au fichier journal.txt avec la date et le message.
  3. Format : [2024-01-15 14:30] Message de l'utilisateur
  4. Affichez le contenu du journal sous le formulaire en utilisant file().
  5. Ajoutez un bouton "Vider le journal" qui efface le fichier (avec confirmation).
  6. Utilisez file_put_contents avec FILE_APPEND pour l'écriture.
EX 02 🖼️ Galerie d'images
  1. Créez galerie.php avec un formulaire d'upload (images uniquement).
  2. Validez : type MIME (JPEG/PNG/GIF), taille max 2 Mo, dimensions max 2000×2000.
  3. Générez un nom unique avec uniqid() et conservez l'extension.
  4. Affichez toutes les images du dossier uploads/ dans une grille.
  5. Ajoutez une gestion d'erreur : fichiers trop grands, mauvais type, etc.
  6. Affichez chaque image avec sa taille (en Ko) et son nom.

Bilan & glossaire

Ce que vous maîtrisez

  • fopen/fclose avec les 6 modes (r, w, a, r+, w+, a+)
  • fwrite pour écrire, fgets pour lire ligne par ligne
  • file() pour lire tout le fichier dans un tableau
  • file_get_contents / file_put_contents pour les opérations simples
  • $_FILES et la structure d'un upload
  • move_uploaded_file pour déplacer un fichier uploadé
  • Validation : type MIME réel, taille, extension, dimensions
  • Gestion des codes d'erreur d'upload

Mini-glossaire

fopen
Ouvre un fichier et retourne un pointeur de ressource.
fclose
Ferme un pointeur de fichier ouvert par fopen.
fwrite
Écrit une chaîne dans un fichier ouvert.
fgets
Lit une ligne depuis un fichier ouvert.
file_get_contents
Lit tout un fichier dans une chaîne (alternative moderne à fopen/fgets).
file_put_contents
Écrit une chaîne dans un fichier (alternative moderne à fopen/fwrite).
$_FILES
Tableau superglobal contenant les fichiers uploadés.
move_uploaded_file
Déplace un fichier uploadé depuis le dossier temporaire.
MIME type
Type de fichier réel détecté par finfo (ex: image/jpeg).
path traversal
Attaque utilisant ../ pour accéder à des fichiers hors du dossier autorisé.
