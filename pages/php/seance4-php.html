01

La boucle for

🌍 Navigateur → 📨 Formulaire POST
📨 Formulaire POST → 🖥️ PHP ($_POST)
🖥️ PHP ($_POST) → 📄 Réponse HTML

La boucle for répète un bloc d'instructions un nombre déterminé de fois. Elle est idéale quand vous connaissez à l'avance le nombre d'itérations.

PHP — Boucle for 
<?php
// Syntaxe : for (initialisation; condition; incrément)
for ($i = 0; $i < 5; $i++) {
    echo "Itération n° $i<br>";
}
// Affiche : Itération n° 0, 1, 2, 3, 4

// Parcourir un tableau indexé
$fruits = ["pomme", "banane", "cerise"];
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "<br>";
}

// Compter à rebours
for ($i = 10; $i > 0; $i--) {
    echo "$i... ";
}
echo "Décollage !";

// Pas de 2
for ($i = 0; $i <= 10; $i += 2) {
    echo "$i ";   // 0 2 4 6 8 10
}
?>
02

La boucle while

while répète un bloc tant que la condition est vraie. Elle est utile quand le nombre d'itérations n'est pas connu à l'avance.

PHP — Boucle while 
<?php
$i = 0;
while ($i < 5) {
    echo "Tour n° $i<br>";
    $i++;
}

// Lecture d'un fichier ligne par ligne
$fichier = fopen("data.txt", "r");
while (($ligne = fgets($fichier)) !== false) {
    echo $ligne . "<br>";
}
fclose($fichier);

// Attention : boucle infinie !
// while (true) { echo "jamais arrêt"; }
// Utilisez break pour en sortir
?>
⚠️
Boucle infinie

Oublier d'incrémenter la variable de condition ($i++) crée une boucle infinie qui fera planter votre script (timeout 30s par défaut). Si ça arrive, stoppez Apache et corrigez le code.

03

La boucle do-while

do-while est similaire à while, mais la condition est vérifiée après l'exécution du bloc. Le bloc s'exécute donc au moins une fois.

PHP — Boucle do-while 
<?php
$i = 0;
do {
    echo "Tour n° $i<br>";
    $i++;
} while ($i < 5);

// Différence clé avec while :
$i = 100;
while ($i < 5) {
    echo "Jamais exécuté";      // condition fausse dès le départ
}

do {
    echo "Exécuté UNE fois";    // exécuté avant la vérification
    $i++;
} while ($i < 5);
?>
💡
Quand utiliser do-while ?

Les interfaces utilisateur où vous voulez au moins une interaction avant de vérifier : validation de formulaire, menu répétitif, génération aléatoire avec vérification.

foreach — Tableaux indexés

foreach est la boucle la plus pratique pour parcourir les tableaux en PHP. Elle itère sur chaque élément sans avoir à gérer d'index manuellement.

PHP — foreach sur tableau indexé 
<?php
$fruits = ["pomme", "banane", "cerise"];

// foreach simple
foreach ($fruits as $fruit) {
    echo "$fruit<br>";
}

// foreach avec index
foreach ($fruits as $index => $fruit) {
    echo "$index : $fruit<br>";
}
// 0 : pomme
// 1 : banane
// 2 : cerise

// foreach ne modifie pas le tableau original
foreach ($fruits as $fruit) {
    $fruit = strtoupper($fruit);
}
print_r($fruits);   // inchangé

// Pour modifier : utilisez & (référence)
foreach ($fruits as &$fruit) {
    $fruit = strtoupper($fruit);
}
unset($fruit);      // détruit la référence après la boucle
print_r($fruits);   // ["POMME", "BANANE", "CERISE"]
?>

05

foreach — Tableaux associatifs

Avec les tableaux associatifs, foreach donne accès à la clé et à la valeur.

PHP — foreach sur tableau associatif 
<?php
$personne = [
    "nom"    => "Dupont",
    "prenom" => "Alice",
    "age"    => 30,
    "ville"  => "Paris"
];

// Accès aux deux : clé et valeur
foreach ($personne as $cle => $valeur) {
    echo "$cle : $valeur<br>";
}

// Affiche :
// nom : Dupont
// prenom : Alice
// age : 30
// ville : Paris

// Générer une liste HTML
echo "<ul>";
foreach ($personne as $cle => $valeur) {
    echo "<li><strong>" . ucfirst($cle) . "</strong> : $valeur</li>";
}
echo "</ul>";
?>

06

break et continue

break stoppe la boucle immédiatement. continue passe à l'itération suivante sans exécuter le reste du bloc.

PHP — break et continue 
<?php
// break — sort de la boucle
$notes = [12, 15, 8, 19, 7];
foreach ($notes as $note) {
    if ($note < 10) {
        echo "Première note éliminatoire : $note";
        break;   // on arrête tout
    }
    echo "Note valide : $note<br>";
}

// continue — saute l'itération courante
foreach ($notes as $note) {
    if ($note < 10) {
        continue;   // on ignore cette note
    }
    echo "Note >= 10 : $note<br>";
}

// break avec niveau (boucles imbriquées)
for ($i = 0; $i < 5; $i++) {
    for ($j = 0; $j < 5; $j++) {
        if ($i == 2 && $j == 2) {
            break 2;   // sort des DEUX boucles
        }
        echo "($i, $j) ";
    }
}
?>

Tableaux indexés

Un tableau indexé stocke des éléments accessibles par un index numérique (commençant à 0).

PHP — Tableaux indexés 
<?php
// Création
$fruits = ["pomme", "banane", "cerise"];
$notes = array(12, 15, 18, 10);   // syntaxe ancienne

// Ajouter des éléments
$fruits[] = "datte";          // ajoute à la fin (index 3)
array_push($fruits, "fraise"); // idem
$fruits[10] = "kiwi";         // crée un énorme trou !

// Compter
echo count($fruits);           // nombre d'éléments

// Trier
sort($fruits);                 // ordre alphabétique croissant
rsort($fruits);                // ordre décroissant

// Rechercher
echo in_array("banane", $fruits);     // true
echo array_search("banane", $fruits); // index (1)

// Autres fonctions utiles
print_r(array_reverse($fruits));
print_r(array_slice($fruits, 1, 2));  // extrait 2 éléments à partir de l'index 1
$fusion = array_merge($fruits, $notes);
?>

08

Tableaux associatifs

Un tableau associatif utilise des clés nommées (strings) au lieu d'index numériques.

PHP — Tableaux associatifs 
<?php
// Création
$produit = [
    "nom"     => "Casque Audio",
    "prix"    => 79.99,
    "stock"   => 15,
    "couleur" => "noir"
];

// Accès
echo $produit["nom"];      // Casque Audio
echo $produit["prix"];     // 79.99

// Clés et valeurs
$cles = array_keys($produit);   // ["nom", "prix", "stock", "couleur"]
$vals = array_values($produit); // ["Casque Audio", 79.99, 15, "noir"]

// Vérifier si une clé existe
var_dump(array_key_exists("stock", $produit));  // true

// Fusionner (attention : les clés identiques sont écrasées)
$infos = ["marque" => "Sony", "garantie" => "2 ans"];
$complet = array_merge($produit, $infos);

// Tableau multidimensionnel
$catalogue = [
    ["nom" => "Casque", "prix" => 79.99],
    ["nom" => "Clavier", "prix" => 49.99],
    ["nom" => "Souris", "prix" => 29.99]
];

foreach ($catalogue as $article) {
    echo $article["nom"] . " : " . $article["prix"] . " €<br>";
}
?>
💡
array_merge vs + (union)

array_merge() réindexe les tableaux numériques et écrase les clés identiques. L'opérateur + conserve la première clé rencontrée et ne réindexe pas.

Exercices pratiques

EX 01 ✏️ Table de multiplication
  1. Créez un fichier table.php.
  2. Utilisez une boucle for pour afficher la table de multiplication du chiffre 7 (de 1 à 10).
  3. Affichez le résultat formaté : "7 × 3 = 21" sur chaque ligne.
  4. Réécrivez la même logique avec while.
  5. Créez un tableau de notes [12, 15, 8, 19, 7, 14]. Utilisez foreach pour : afficher chaque note, calculer la moyenne, trouver la note maximale.
EX 02 🗂️ Catalogue produits
  1. Créez un tableau associatif $catalogue avec 3 produits (nom, prix, stock).
  2. Affichez chaque produit dans une liste HTML avec foreach.
  3. Ajoutez un nouveau produit avec [].
  4. Filtrez : utilisez continue pour n'afficher que les produits en stock (stock > 0).
  5. Calculez le prix total du stock (prix × stock pour chaque produit).
  6. Utilisez array_merge() pour ajouter une description à chaque produit.

Bilan & glossaire

Ce que vous maîtrisez

  • Les 4 types de boucles : for, while, do-while, foreach
  • break pour sortir d'une boucle, continue pour sauter une itération
  • Les tableaux indexés : création, ajout (array_push, []), tri, count
  • Les tableaux associatifs : clés nommées, array_keys, array_values
  • Les fonctions utiles : in_array, array_search, array_merge, array_slice
  • Les tableaux multidimensionnels

Mini-glossaire

Boucle for
Itération avec compteur : for ($i=0; $i<n; $i++)
Boucle while
Répétition tant qu'une condition est vraie.
Boucle do-while
Exécute le bloc au moins une fois avant de vérifier la condition.
foreach
Boucle spécialisée pour parcourir les éléments d'un tableau.
break
Mot-clé qui interrompt immédiatement la boucle courante.
continue
Mot-clé qui passe à l'itération suivante sans finir le bloc.
Tableau indexé
Collection d'éléments accessibles par index numérique (0, 1, 2…).
Tableau associatif
Collection de paires clé → valeur avec clés nommées.
array_merge
Fusionne deux ou plusieurs tableaux en un seul.
Tableau multidimensionnel
Tableau contenant d'autres tableaux (matrice, catalogue).
