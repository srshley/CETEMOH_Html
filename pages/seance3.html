01

Hiérarchie des titres h1 à h6

HTML propose 6 niveaux de titres, de <h1> (le plus important) à <h6> (le moins important). Ils forment un plan de document comme les chapitres d'un livre.

💡
Règle d'or

Un seul <h1> par page. Ne sautez jamais de niveau : h1 → h2 → h3, jamais h1 → h3. Google utilise cette hiérarchie pour comprendre la structure de votre contenu.

HTML — Hiérarchie des titres 
<h1>Titre principal de la page</h1>
<h2>Chapitre 1</h2>
<h3>Sous-chapitre 1.1</h3>
<h4>Sous-sous-chapitre 1.1.1</h4>
<h2>Chapitre 2</h2>
<h3>Sous-chapitre 2.1</h3>
<h3>Sous-chapitre 2.2</h3>
RENDU

Titre principal de la page

Chapitre 1

Sous-chapitre 1.1

Sous-sous-chapitre 1.1.1

Chapitre 2

Sous-chapitre 2.1

Sous-chapitre 2.2

⚠️
Ne pas utiliser les titres pour la taille du texte

Utilisez <h1> pour le titre principal, pas parce que vous voulez du gros texte. Le CSS (font-size) est fait pour la taille. Les titres sont pour la structure.

02

Paragraphes & sauts de ligne

La balise <p> crée un paragraphe. Le navigateur ajoute automatiquement un espacement vertical entre deux paragraphes. Pour un simple retour à la ligne sans nouveau paragraphe, on utilise <br>.

HTML — Paragraphes et sauts 
<p>Ceci est un paragraphe normal.
Le navigateur ignore le saut de ligne dans le code.</p>

<p>Ceci est un deuxième paragraphe
qui parle d'autre chose.</p>

<p>12 rue des Fleurs<br>
75001 Paris<br>
France</p>
RENDU

Ceci est un paragraphe normal. Le navigateur ignore le saut de ligne dans le code.

Ceci est un deuxième paragraphe qui parle d'autre chose.

12 rue des Fleurs
75001 Paris
France

03

Mise en valeur du texte

HTML offre plusieurs balises pour donner du sens à des portions de texte. Elles ne sont pas juste décoratives : elles ont une signification pour les moteurs de recherche et les lecteurs d'écran.

HTML — strong, em, mark 
<p>Le <strong>HTML</strong> est le fondement du web.</p>
<p>J'<em>adore</em> coder en HTML !</p>
<p><mark>À retenir</mark> : toute balise se ferme.</p>
<p>Appuyez sur <kbd>Ctrl</kbd> + <kbd>S</kbd> pour sauvegarder.</p>
<p>Le code <code>&lt;p&gt;</code> crée un paragraphe.</p>
RENDU

Le HTML est le fondement du web.

J'adore coder en HTML !

À retenir : toute balise se ferme.

Appuyez sur Ctrl + S pour sauvegarder.

Le code <p> crée un paragraphe.

04

Citations

Trois balises permettent de gérer les citations : <blockquote> pour les citations longues, <q> pour les citations courtes, et <cite> pour la source.

HTML — Citations 
<!-- Citation longue -->
<blockquote cite="https://fr.wikiquote.org">
  <p>"La créativité, c'est l'intelligence qui s'amuse."</p>
  <footer>— Albert Einstein</footer>
</blockquote>

<!-- Citation courte -->
<p>Il a dit : <q>Le HTML est facile !</q></p>

<!-- Source de citation -->
<p>Voir <cite>HTML5 for Web Designers</cite> de Jeremy Keith.</p>
RENDU

"La créativité, c'est l'intelligence qui s'amuse."

— Albert Einstein

Il a dit : Le HTML est facile !

Voir HTML5 for Web Designers de Jeremy Keith.

05

Code & préformaté

Pour afficher du code source, HTML fournit plusieurs balises selon le contexte : <code> pour le code inline, <pre> pour conserver les espaces et retours à la ligne, <kbd> pour les touches clavier, et <samp> pour la sortie d'un programme.

HTML — Code et préformaté 
<!-- Code inline -->
<p>Utilisez <code>&lt;p&gt;</code> pour un paragraphe.</p>

<!-- Préformaté (conserve espaces et retours) -->
<pre>
  function hello() {
    console.log("Bonjour !");
  }
</pre>

<!-- Combinaison code + pre pour du code source -->
<pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="fr"&gt;
  &lt;head&gt;
    &lt;title&gt;Exemple&lt;/title&gt;
  &lt;/head&gt;
&lt;/html&gt;
</code></pre>

<!-- Samp : sortie d'un programme -->
<p><samp>Erreur 404 : fichier non trouvé</samp></p>
RENDU

Utilisez <p> pour un paragraphe.

  function hello() {
    console.log("Bonjour !");
  }

06

Abréviations & Adresses

<abbr> permet de marquer une abréviation ou un acronyme avec l'attribut title pour la signification complète. <address> sert à marquer les coordonnées de l'auteur ou du site.

HTML — Abbr et Address 
<!-- Abréviation avec infobulle -->
<p>Le <abbr title="HyperText Markup Language">HTML</abbr>
est le langage du web.</p>

<p>L'<abbr title="Organisation des Nations Unies">ONU</abbr>
a été fondée en 1945.</p>

<!-- Adresse de contact -->
<address>
  <strong>Jean Dupont</strong><br>
  12 rue des Fleurs<br>
  75001 Paris<br>
  France<br>
  <a href="mailto:jean@exemple.com">jean@exemple.com</a>
</address>
RENDU

Le HTML est le langage du web.

L'ONU a été fondée en 1945.

Jean Dupont
12 rue des Fleurs
75001 Paris
France
jean@exemple.com
07

Contenu modifié : del, ins, small

Ces balises indiquent des modifications dans le texte : <del> pour ce qui a été supprimé, <ins> pour ce qui a été ajouté, <small> pour les notes secondaires.

HTML — del, ins, small 
<!-- Texte supprimé et ajouté -->
<p>Prix : <del>100&nbsp;€</del>
<ins>80&nbsp;€</ins> (promotion !)</p>

<!-- Note secondaire -->
<p><small>&copy; 2024 Mon Site.
Tous droits réservés.</small></p>

<!-- Combinaison -->
<p>Ancienne version : <del>HTML 4.01</del>
<ins>HTML5</ins></p>
<p><small>Mis à jour le 15 mars 2024</small></p>
RENDU

Prix : 100 € 80 € (promotion !)

© 2024 Mon Site. Tous droits réservés.

Ancienne version : HTML 4.01 HTML5

Mis à jour le 15 mars 2024

08

Séparateur & entités spéciales

La balise orpheline <hr> (horizontal rule) crée une séparation thématique entre deux sections. Combinée aux entités HTML, elle permet de structurer proprement le contenu.

HTML — hr et entités 
<h2>Section 1</h2>
<p>Contenu de la première section.</p>

<hr>

<h2>Section 2</h2>
<p>Contenu de la deuxième section.</p>

<hr>

<!-- Entités spéciales pour la typographie -->
<p>Guillemets français : &laquo; Bonjour &raquo;</p>
<p>Tiret cadratin : &mdash; pour les incises</p>
<p>Apostrophe typographique : &rsquo; (plutôt que ')</p>
<p>Multiplication : 3 &times; 4 = 12</p>
<p>Exposant : m&sup2; (mètre carré)</p>
RENDU

Section 1

Contenu de la première section.

Section 2

Contenu de la deuxième section.

Guillemets français : « Bonjour »

Tiret cadratin : — pour les incises

Apostrophe typographique : ’ (plutôt que ')

Multiplication : 3 × 4 = 12

Exposant : m² (mètre carré)

09

Exercices pratiques

EX 01 ✏️ Page avec structure de titres
  1. Créez seance3-exo1.html avec la structure HTML5.
  2. Utilisez un <h1> "Ma page de présentation".
  3. Ajoutez 3 sections avec <h2> : "Qui suis-je", "Mes compétences", "Mes hobbies".
  4. Chaque section doit avoir au moins un <h3>.
  5. Utilisez <strong>, <em> et <mark> dans les paragraphes.
EX 02 ✏️ Article de blog avec citations
  1. Écrivez un court article sur un sujet qui vous passionne.
  2. Incluez une citation longue avec <blockquote> et une citation courte avec <q>.
  3. Ajoutez un <hr> entre l'introduction et le corps.
  4. Utilisez <abbr> pour au moins deux acronymes.
  5. Ajoutez une adresse de contact avec <address>.
EX 03 🔧 Code et entités
  1. Créez une page qui explique une balise HTML (par exemple <a>).
  2. Affichez la balise littéralement en utilisant &lt; et &gt;.
  3. Utilisez <pre><code> pour montrer un exemple de code.
  4. Utilisez <kbd> pour les raccourcis clavier.
  5. Utilisez <samp> pour une sortie de programme.
EX 04 🔄 Contenu modifié
  1. Créez une page "Notes de version" (changelog).
  2. Utilisez <del> pour les fonctionnalités supprimées.
  3. Utilisez <ins> pour les nouvelles fonctionnalités.
  4. Utilisez <small> pour les notes de bas de page.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Hiérarchie des titres h1-h6 avec règle du h1 unique
  • Paragraphes <p> et sauts de ligne <br>
  • Balises de mise en valeur : strong, em, mark
  • Citations : blockquote, q, cite
  • Code : code, pre, kbd, samp
  • Abréviations abbr et adresse address
  • Contenu modifié : del, ins, small
  • Séparateur hr et entités typographiques

Mini-glossaire

Hiérarchie des titres
Ordre logique h1→h6. Un seul h1, jamais de saut de niveau.
Paragraphe
Bloc de texte créé par <p>. Espacement automatique entre les paragraphes.
<strong>
Importance forte. Rendu en gras mais sens ≠ style.
<em>
Emphase. Rendu en italique. Change l'intonation pour les lecteurs d'écran.
<blockquote>
Citation longue d'une source externe. Peut contenir d'autres éléments.
<pre>
Texte préformaté. Conserve les espaces et retours à la ligne exacts.
<abbr>
Marque une abréviation avec son expansion dans l'attribut title.
<del> / <ins>
Indiquent respectivement du contenu supprimé ou ajouté.
<hr>
Séparation thématique entre sections. Balise orpheline.
Entité typographique
Code pour les caractères spéciaux : &laquo;, &raquo;, &times;, &sup2;.
