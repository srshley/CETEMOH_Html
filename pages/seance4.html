01

Listes non ordonnées <ul>

La liste non ordonnée (<ul>) affiche des éléments précédés de puces. Utilisez-la quand l'ordre n'a pas d'importance : liste de courses, menu de navigation, caractéristiques d'un produit.

HTML — Liste non ordonnée 
<h2>Liste de courses</h2>
<ul>
  <li>Pain</li>
  <li>Lait</li>
  <li>Œufs</li>
  <li>Beurre</li>
  <li>Fromage</li>
</ul>
RENDU

Liste de courses

  • Pain
  • Lait
  • Œufs
  • Beurre
  • Fromage
💡
Accessibilité

Les lecteurs d'écran annoncent "liste de N éléments" pour une vraie <ul>. N'utilisez pas des tirets simulés <p>- Pain</p>.

02

Listes ordonnées <ol>

La liste ordonnée (<ol>) numérote automatiquement ses éléments. Utilisez-la quand l'ordre est important : recette de cuisine, top 10, marche à suivre.

HTML — Liste ordonnée 
<h2>Préparation</h2>
<ol>
  <li>Préchauffer le four à 180°C</li>
  <li>Mélanger farine, sucre et beurre</li>
  <li>Ajouter les œufs un par un</li>
  <li>Verser dans un moule</li>
  <li>Enfourner 25 minutes</li>
</ol>
RENDU

Préparation

  1. Préchauffer le four à 180°C
  2. Mélanger farine, sucre et beurre
  3. Ajouter les œufs un par un
  4. Verser dans un moule
  5. Enfourner 25 minutes

Attributs avancés de <ol>

AttributExempleEffet
typetype="A"A, B, C… (1, a, A, i, I)
startstart="5"Commence à 5
reversedreversedOrdre décroissant
HTML — Attributs de ol 
<!-- Type A (majuscules) -->
<ol type="A">
  <li>Premier</li>
  <li>Deuxième</li>
  <li>Troisième</li>
</ol>

<!-- Start à 5 -->
<ol start="5">
  <li>Cinq</li>
  <li>Six</li>
</ol>

<!-- Ordre inverse -->
<ol reversed>
  <li>Dernier</li>
  <li>Avant-dernier</li>
</ol>
RENDU
  1. Premier
  2. Deuxième
  3. Troisième
  1. Cinq
  2. Six
  1. Dernier
  2. Avant-dernier
03

Imbrication de listes

Un élément <li> peut lui-même contenir une liste. Cela permet de créer des hiérarchies comme un plan de cours, un arbre de catégories, ou un menu déroulant.

HTML — Listes imbriquées 
<ul>
  <li>Fruits
    <ul>
      <li>Pomme</li>
      <li>Banane</li>
      <li>Orange</li>
    </ul>
  </li>
  <li>Légumes
    <ul>
      <li>Carotte</li>
      <li>Brocoli</li>
    </ul>
  </li>
</ul>
RENDU
  • Fruits
    • Pomme
    • Banane
    • Orange
  • Légumes
    • Carotte
    • Brocoli
04

Listes de définition <dl>

<dl> (description list) associe des termes (<dt>) à leurs définitions (<dd>). Parfaite pour les glossaires, les métadonnées, ou les paires clé/valeur.

HTML — Liste de définition 
<dl>
  <dt>HTML</dt>
  <dd>Langage de balisage pour structurer le contenu web.</dd>

  <dt>CSS</dt>
  <dd>Langage de style pour mettre en forme les pages.</dd>

  <dt>JavaScript</dt>
  <dd>Langage de programmation pour les interactions.</dd>
</dl>
RENDU
HTML
Langage de balisage pour structurer le contenu web.
CSS
Langage de style pour mettre en forme les pages.
JavaScript
Langage de programmation pour les interactions.
05

Liens externes

La balise <a> (anchor) crée un lien hypertexte. Pour un lien vers un autre site, l'attribut href doit contenir l'URL complète avec https://.

HTML — Liens externes 
<!-- Lien externe vers un autre site -->
<a href="https://www.wikipedia.org">
  Aller sur Wikipedia
</a>

<!-- Lien avec target _blank (nouvel onglet) -->
<a href="https://exemple.com"
   target="_blank"
   rel="noopener noreferrer">
  Ouvrir dans un nouvel onglet
</a>
RENDU

Aller sur Wikipedia

Ouvrir dans un nouvel onglet

⚠️
Sécurité — rel="noopener noreferrer"

Sans rel="noopener", la page ouverte peut rediriger votre page vers un site malveillant (attaque tabnapping). Ajoutez toujours ces deux attributs avec target="_blank".

06

Liens internes & chemins relatifs

Un lien interne pointe vers une autre page du même site. On utilise des chemins relatifs calculés depuis le fichier courant.

Structure d'un site typique :

📁mon-site/
📄index.html← fichier courant
📄contact.htmlhref="contact.html"
📁blog/
📄article.htmlhref="blog/article.html"
📁images/
🖼️logo.pngsrc="images/logo.png"
HTML — Chemins relatifs 
<!-- Même dossier -->
<a href="contact.html">Contact</a>

<!-- Sous-dossier -->
<a href="blog/article.html">Article</a>

<!-- Dossier parent (../) -->
<a href="../index.html">Retour à l'accueil</a>

<!-- Racine du site -->
<a href="/index.html">Accueil absolu</a>
RENDU

Contact · Article · Retour à l'accueil · Accueil absolu

07

Ancres #id

Un lien peut pointer vers une section spécifique de la même page en utilisant le # suivi de l'attribut id de la cible. Le navigateur scroll automatiquement jusqu'à l'élément.

HTML — Ancres 
<!-- Menu de navigation avec ancres -->
<nav>
  <a href="#presentation">Présentation</a>
  <a href="#competences">Compétences</a>
  <a href="#contact">Contact</a>
</nav>

<!-- Sections cibles avec id -->
<h2 id="presentation">Présentation</h2>
<p>Contenu de la présentation...</p>

<h2 id="competences">Compétences</h2>
<p>Liste des compétences...</p>

<h2 id="contact">Contact</h2>
<p>Formulaire de contact...</p>

<!-- Retour en haut -->
<a href="#">↑ Retour en haut de page</a>

<!-- Ancre sur une autre page -->
<a href="contact.html#formulaire">
  Aller au formulaire sur la page contact
</a>
RENDU
PrésentationCompétencesContact

↑ Retour en haut de page

08

Attributs target, rel, download, mailto, tel

La balise <a> accepte plusieurs attributs qui modifient le comportement du lien.

HTML — Tous les types de liens 
<!-- Nouvel onglet sécurisé -->
<a href="https://exemple.com"
   target="_blank"
   rel="noopener noreferrer">
  Ouvrir dans un nouvel onglet
</a>

<!-- Email -->
<a href="mailto:contact@exemple.com">
  Nous écrire
</a>
<a href="mailto:contact@exemple.com?
  subject=Demande&body=Bonjour">
  Email avec sujet
</a>

<!-- Téléphone (mobile) -->
<a href="tel:+33600000000">
  +33 6 00 00 00 00
</a>

<!-- Téléchargement -->
<a href="guide.pdf" download="Guide_HTML.pdf">
  Télécharger le guide
</a>
RENDU

Ouvrir dans un nouvel onglet

Nous écrire · Email avec sujet

+33 6 00 00 00 00 · Télécharger le guide

AttributValeursRôle
target_blank, _self, _parent, _topOù ouvrir le lien
relnoopener, noreferrer, nofollowRelation entre pages
downloadNom de fichierForce le téléchargement
mailto:Adresse emailOuvre le client mail
tel:Numéro internationalCompose sur mobile
09

Exercices pratiques

EX 01 ✏️ Page avec listes
  1. Créez une page "Mes recettes préférées".
  2. Utilisez une <ul> pour lister 5 recettes.
  3. Dans chaque item, ajoutez une <ol> pour les étapes.
  4. Ajoutez une <dl> pour définir les termes de cuisine.
EX 02 🔗 Navigation complète
  1. Créez 3 fichiers HTML liés entre eux : index.html, about.html, contact.html.
  2. Chaque page a un menu de navigation vers les 2 autres.
  3. Ajoutez un lien externe vers Google avec target="_blank" et rel="noopener".
  4. Ajoutez un mailto: et un tel:.
EX 03 ⚓ Ancres
  1. Créez une page "FAQ" avec 5 questions-réponses.
  2. En haut, un menu avec des ancres vers chaque question.
  3. À côté de chaque réponse, un lien "↑ Retour" pointant vers le menu.
  4. Ajoutez scroll-behavior: smooth en CSS pour l'animation.

10

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Listes non ordonnées ul avec puces
  • Listes ordonnées ol : type (1, A, a, I, i), start, reversed
  • Imbrication de listes (ul dans ul, ol dans li)
  • Listes de définition dl, dt, dd
  • Liens externes avec https:// et target="_blank" sécurisé
  • Liens internes et chemins relatifs (./ ../ /)
  • Ancres #id pour la navigation intra-page
  • Attributs : target, rel, download, mailto:, tel:

Mini-glossaire

ul
Liste non ordonnée — puces. Ordre non important.
ol
Liste ordonnée — numérotation. Ordre important.
li
Élément de liste. Peut contenir une sous-liste.
dl / dt / dd
Liste de définition : terme (dt) et définition (dd).
<a>
Balise ancre pour les liens hypertextes.
href
Attribut de destination du lien. URL, #id, mailto:, tel:.
Chemin relatif
Calculé depuis le fichier courant. Ex: "pages/contact.html".
target="_blank"
Ouvre le lien dans un nouvel onglet. Toujours avec rel="noopener".
Ancre
Lien interne vers une section via #id. Scroll automatique.
