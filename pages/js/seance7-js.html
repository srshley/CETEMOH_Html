01

addEventListener — écouter les événements

🖱️ Clic utilisateur → 📡 Événement DOM
📡 Événement DOM → ⚡ addEventListener
⚡ addEventListener → 🎯 Fonction callback
🎯 Fonction callback → ✨ DOM mis à jour

addEventListener est la méthode moderne pour réagir aux interactions utilisateur.

JavaScript 
const btn = document.getElementById('demo-btn');

// Syntaxe : element.addEventListener('event', callback)
btn.addEventListener('click', function() {
  alert('Bouton cliqué !');
});

// Avec fonction fléchée
btn.addEventListener('click', () => {
  console.log('Clic détecté');
});

// Plusieurs écouteurs sur le même élément
btn.addEventListener('click', () => console.log('1'));
btn.addEventListener('click', () => console.log('2'));
// Les deux s'exécutent dans l'ordre

// Options : { once: true } → une seule fois
btn.addEventListener('click', () => {
  console.log('Exécuté une seule fois');
}, { once: true });

02

Événements click et souris

Les événements de souris les plus courants.

JavaScript 
const zone = document.getElementById('demo-mouse');

// click — clic complet (mousedown + mouseup)
zone.addEventListener('click', () => {
  console.log('Clic');
});

// dblclick — double-clic
zone.addEventListener('dblclick', () => {
  console.log('Double clic');
});

// mouseenter — souris entre dans l'élément (sans bouillonnement)
zone.addEventListener('mouseenter', () => {
  zone.style.background = '#e0f0ff';
});

// mouseleave — souris quitte l'élément
zone.addEventListener('mouseleave', () => {
  zone.style.background = '';
});

// mousemove — souris bouge (coordonnées)
zone.addEventListener('mousemove', (e) => {
  console.log(`X: ${e.offsetX}, Y: ${e.offsetY}`);
});

// contextmenu — clic droit
zone.addEventListener('contextmenu', (e) => {
  e.preventDefault();
  console.log('Menu contextuel bloqué');
});
03

Événements input et change

Réagir à la saisie utilisateur dans les champs de formulaire.

JavaScript 
const input = document.getElementById('demo-input');
const affichage = document.getElementById('demo-aff');

// input — se déclenche à chaque saisie
input.addEventListener('input', () => {
  affichage.textContent = input.value || '(vide)';
});

// change — se déclenche quand le champ perd le focus
input.addEventListener('change', () => {
  console.log('Valeur finale:', input.value);
});

// focus — champ reçoit le focus
input.addEventListener('focus', () => {
  input.style.borderColor = '#3B82F6';
});

// blur — champ perd le focus
input.addEventListener('blur', () => {
  input.style.borderColor = '#ccc';
});

// select — changement de sélection (select/checkbox/radio)
const select = document.getElementById('demo-select');
select.addEventListener('change', () => {
  affichage.textContent = `Vous avez choisi : ${select.value}`;
});

04

Événement submit — formulaires

Intercepter la soumission d'un formulaire pour la traiter en JS.

JavaScript 
const form = document.getElementById('demo-form');
const result = document.getElementById('demo-result');

form.addEventListener('submit', (e) => {
  // Empêche le rechargement de la page
  e.preventDefault();

  // Récupérer les valeurs
  const prenom = form.prenom.value.trim();
  const email = form.email.value.trim();

  // Validation simple
  if (!prenom || !email) {
    result.textContent = 'Veuillez remplir tous les champs';
    result.style.color = 'red';
    return;
  }

  // Afficher le résultat
  result.innerHTML = `
    ✅ Merci ${prenom} !
    Nous vous contacterons à ${email}.
  `;
  result.style.color = '#155724';

  // Réinitialiser
  form.reset();
});

05

L'objet événement

Le callback reçoit un objet event avec des informations sur l'événement.

JavaScript — Event object 
const zone = document.getElementById('demo-event-obj');

zone.addEventListener('click', (e) => {
  // Type d'événement
  console.log(e.type); // "click"

  // Élément qui a déclenché l'événement
  console.log(e.target);     // l'élément cliqué
  console.log(e.currentTarget); // élément qui écoute (ici, zone)

  // Coordonnées
  console.log(`Page: ${e.pageX}, ${e.pageY}`);
  console.log(`Client: ${e.clientX}, ${e.clientY}`);
  console.log(`Offset: ${e.offsetX}, ${e.offsetY}`);

  // Bouton de la souris
  console.log(e.button); // 0 = gauche, 1 = molette, 2 = droit

  // Touches modificateurs
  if (e.ctrlKey) console.log('Ctrl pressé');
  if (e.shiftKey) console.log('Shift pressé');
  if (e.altKey) console.log('Alt pressé');

  // Empêcher le comportement par défaut
  e.preventDefault();

  // Arrêter la propagation
  e.stopPropagation();
});
06

Délégation d'événements

La délégation consiste à écouter un événement sur un parent plutôt que sur chaque enfant. Utile pour les éléments créés dynamiquement.

JavaScript 
const liste = document.getElementById('demo-liste');
const ajouter = document.getElementById('demo-ajouter');

// Délégation : on écoute sur le parent <ul>
liste.addEventListener('click', (e) => {
  const li = e.target.closest('li');
  if (!li) return; // ignorer si pas un <li>

  console.log(`Clic sur : ${li.textContent}`);

  // Supprimer l'élément
  if (e.target.classList.contains('suppr')) {
    li.remove();
  }
});

// Ajouter dynamiquement
ajouter.addEventListener('click', () => {
  const li = document.createElement('li');
  li.innerHTML = `Item ${liste.children.length + 1} `;
  liste.appendChild(li);
});

07

Événements clavier

Détecter les touches pressées par l'utilisateur.

JavaScript 
// keydown — touche enfoncée
document.addEventListener('keydown', (e) => {
  console.log(`keydown: ${e.key} (code: ${e.code})`);

  // Touches spéciales
  if (e.key === 'Escape') {
    console.log('Échap pressé — fermer la modale');
  }

  // Raccourcis clavier
  if (e.ctrlKey && e.key === 's') {
    e.preventDefault();
    console.log('Sauvegarde rapide (Ctrl+S)');
  }

  // Flèches
  if (e.key === 'ArrowUp') console.log('↑');
  if (e.key === 'ArrowDown') console.log('↓');
  if (e.key === 'ArrowLeft') console.log('←');
  if (e.key === 'ArrowRight') console.log('→');
});

// keyup — touche relâchée
document.addEventListener('keyup', (e) => {
  console.log(`keyup: ${e.key}`);
});

// Différence key vs code :
// key = caractère produit ('a', 'A', 'é')
// code = touche physique ('KeyA')
// Sur un clavier QWERTY presser 'w' → key='w', code='KeyW'

// keypress — déprécié, ne pas utiliser
08

removeEventListener — retirer un écouteur

On peut retirer un écouteur avec removeEventListener. Nécessite la même fonction référence.

JavaScript 
const btn = document.getElementById('demo-remove-btn');
const toggle = document.getElementById('demo-toggle');

// La fonction doit être nommée (pas une fonction anonyme)
function handleClick() {
  console.log('Bouton cliqué !');
}

// Ajouter
btn.addEventListener('click', handleClick);

// Retirer
toggle.addEventListener('click', () => {
  if (btn._listening) {
    btn.removeEventListener('click', handleClick);
    toggle.textContent = 'Réactiver';
    btn._listening = false;
  } else {
    btn.addEventListener('click', handleClick);
    toggle.textContent = 'Désactiver';
    btn._listening = true;
  }
});

// Important : les fonctions fléchées anonymes
// NE PEUVENT PAS être retirées
// ❌ btn.addEventListener('click', () => { ... });
// ❌ btn.removeEventListener('click', () => { ... }); // ne marche PAS

// Utilisation : dé-bouncer temporairement un clic
function clickLimite() {
  console.log('Clic accepté');
  btn.removeEventListener('click', clickLimite);
  setTimeout(() => {
    btn.addEventListener('click', clickLimite);
  }, 1000);
}

Exercices pratiques

EX 01 🖱️ Interactions souris
  1. Créez une <div> carrée de 200×200px avec fond gris et id="zone".
  2. Au clic : changez la couleur en bleu.
  3. Au double-clic : remettez la couleur grise.
  4. Au survol (mouseenter) : ajoutez une bordure.
  5. Quand la souris quitte (mouseleave) : retirez la bordure.
  6. Affichez les coordonnées de la souris dans la div.
EX 02 📝 Formulaire avec validation
  1. Créez un formulaire avec : nom (texte), email (email), âge (nombre 1-120), et un select (pays).
  2. Ajoutez un submit listener avec preventDefault.
  3. Validez que tous les champs sont remplis.
  4. Validez que l'email contient @.
  5. Validez que l'âge est entre 1 et 120.
  6. Affichez un message de succès ou d'erreur.
  7. Ajoutez un champ avec input événement qui affiche en temps réel le nombre de caractères saisis.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • addEventListener : syntaxe, fonctions nommées et fléchées
  • Événements souris : click, dblclick, mouseenter/leave, mousemove
  • Événements de champ : input (temps réel), change (perte focus)
  • submit : intercepter, preventDefault, récupérer et valider
  • Event object : target, currentTarget, coordonnées, touches modificateurs
  • Délégation d'événements avec closest() sur parent
  • Clavier : keydown, keyup, key vs code, raccourcis
  • removeEventListener : fonction nommée requise

Mini-glossaire

addEventListener
Méthode pour attacher un écouteur d'événement à un élément.
Event object
Objet passé automatiquement au callback. Contient type, target, coordonnées, etc.
preventDefault()
Méthode pour annuler le comportement par défaut (ex: soumission de formulaire).
Délégation
Technique qui consiste à écouter sur un parent plutôt que sur chaque enfant, en utilisant le bouillonnement (bubbling).
Bouillonnement (bubbling)
Propagation d'un événement de l'élément cible vers les parents. Permet la délégation.
target vs currentTarget
target = élément qui a déclenché l'événement. currentTarget = élément qui écoute.
