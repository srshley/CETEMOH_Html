01

Méthodes de mutation : push, pop, shift, unshift

Ces méthodes modifient le tableau directement (elles mutent l'original).

JavaScript — Mutations de tableau 
const fruits = ['pomme', 'banane'];

// push() — ajoute à la fin (retourne la nouvelle longueur)
fruits.push('cerise');
console.log(fruits); // ["pomme", "banane", "cerise"]

// pop() — retire le dernier (retourne l'élément retiré)
const dernier = fruits.pop();
console.log(dernier); // "cerise"
console.log(fruits);  // ["pomme", "banane"]

// unshift() — ajoute au début
fruits.unshift('abricot');
console.log(fruits); // ["abricot", "pomme", "banane"]

// shift() — retire le premier
const premier = fruits.shift();
console.log(premier); // "abricot"
console.log(fruits);  // ["pomme", "banane"]

// splice() — ajoute/supprime à une position
fruits.splice(1, 0, 'kiwi'); // à l'index 1, supprime 0, ajoute kiwi
console.log(fruits); // ["pomme", "kiwi", "banane"]

fruits.splice(1, 1); // supprime 1 élément à l'index 1
console.log(fruits); // ["pomme", "banane"]
02

length, indexOf, includes

Propriétés et méthodes pour interroger un tableau.

JavaScript — Interroger un tableau 
const notes = [12, 15, 8, 19, 10];

// length — nombre d'éléments
console.log(notes.length); // 5

// indexOf() — position d'un élément (-1 si absent)
console.log(notes.indexOf(15)); // 1
console.log(notes.indexOf(20)); // -1

// includes() — true/false si élément présent
console.log(notes.includes(19)); // true
console.log(notes.includes(7));  // false

// lastIndexOf() — dernière occurrence
const nombres = [1, 2, 3, 2, 1];
console.log(nombres.lastIndexOf(2)); // 3

// join() — concatène en chaîne
console.log(notes.join(' - ')); // "12 - 15 - 8 - 19 - 10"

// concat() — fusionne des tableaux (ne mute pas)
const a = [1, 2];
const b = [3, 4];
const c = a.concat(b);
console.log(c); // [1, 2, 3, 4]
console.log(a); // [1, 2] (inchangé)

03

forEach

🖱️ Clic utilisateur → 📡 Événement DOM
📡 Événement DOM → ⚡ addEventListener
⚡ addEventListener → 🎯 Fonction callback
🎯 Fonction callback → ✨ DOM mis à jour

forEach exécute une fonction pour chaque élément du tableau. C'est une boucle fonctionnelle.

JavaScript — forEach 
const fruits = ['pomme', 'banane', 'cerise'];

// Syntaxe de base
fruits.forEach(function(fruit) {
  console.log(fruit);
});

// Avec fonction fléchée
fruits.forEach(fruit => console.log(fruit));

// Avec index
fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});
// → 0: pomme, 1: banane, 2: cerise

// Avec le tableau complet en 3e paramètre
fruits.forEach((fruit, i, arr) => {
  console.log(`${fruit} / ${arr.length} éléments`);
});

// forEach ne retourne rien (undefined)
// Il ne peut pas être chaîné comme map/filter
// On ne peut pas break/continue dans forEach

// Comparaison avec for...of
for (const fruit of fruits) {
  console.log(fruit);
}
// for...of supporte break, continue, await

map — transformer chaque élément

map crée un nouveau tableau avec le résultat de l'appel de fonction sur chaque élément. Ne mute pas l'original.

JavaScript — map 
const nombres = [1, 2, 3, 4, 5];

// Doubler chaque nombre
const doubles = nombres.map(n => n * 2);
console.log(doubles); // [2, 4, 6, 8, 10]
console.log(nombres); // [1, 2, 3, 4, 5] (inchangé)

// Convertir en chaînes
const enChaines = nombres.map(n => `Nombre: ${n}`);
console.log(enChaines);
// → ["Nombre: 1", "Nombre: 2", "Nombre: 3", "Nombre: 4", "Nombre: 5"]

// Transformer des objets
const produits = [
  { nom: 'Clavier', prix: 50 },
  { nom: 'Souris', prix: 30 },
  { nom: 'Écran', prix: 200 }
];

const nomsProduits = produits.map(p => p.nom);
console.log(nomsProduits); // ["Clavier", "Souris", "Écran"]

// Ajouter une propriété
const avecTva = produits.map(p => ({
  ...p,
  prixTTC: p.prix * 1.2
}));
console.log(avecTva);
// → [{ nom: "Clavier", prix: 50, prixTTC: 60 }, ...]

// map avec index
const fruits = ['pomme', 'banane', 'cerise'];
const prefixes = fruits.map((f, i) => `${i + 1}. ${f}`);
console.log(prefixes);
// → ["1. pomme", "2. banane", "3. cerise"]

05

filter — sélectionner des éléments

filter crée un nouveau tableau avec les éléments qui passent un test.

JavaScript — filter 
const notes = [12, 15, 8, 19, 10, 6, 14];

// Garder les notes >= 10
const admis = notes.filter(n => n >= 10);
console.log(admis);  // [12, 15, 19, 10, 14]
console.log(notes);  // [12, 15, 8, 19, 10, 6, 14] (inchangé)

// Garder les nombres pairs
const pairs = notes.filter(n => n % 2 === 0);
console.log(pairs);  // [12, 8, 10, 6, 14]

// Filtrer des objets
const produits = [
  { nom: 'Clavier', prix: 50, stock: true },
  { nom: 'Souris', prix: 30, stock: false },
  { nom: 'Écran', prix: 200, stock: true },
  { nom: 'Casque', prix: 80, stock: false }
];

// Produits en stock
const enStock = produits.filter(p => p.stock);
console.log(enStock);
// → [{ nom: "Clavier", prix: 50, stock: true }, { nom: "Écran", prix: 200, stock: true }]

// Produits à moins de 100€ ET en stock
const promos = produits.filter(p => p.prix < 100 && p.stock);
console.log(promos);
// → [{ nom: "Clavier", prix: 50, stock: true }]

// Filter avec index
const everyOther = notes.filter((_, i) => i % 2 === 0);
console.log(everyOther); // [12, 8, 10, 14]

reduce — réduire à une valeur

reduce exécute un accumulateur sur chaque élément pour réduire le tableau à une seule valeur.

JavaScript — reduce 
const nombres = [10, 20, 30, 40];

// Somme (accumulateur commence à 0)
const somme = nombres.reduce((acc, n) => acc + n, 0);
console.log(somme); // 100

// Accumulateur par défaut = premier élément si non fourni
const somme2 = nombres.reduce((acc, n) => acc + n);
console.log(somme2); // 100 (identique ici)

// Produit
const produit = nombres.reduce((acc, n) => acc * n, 1);
console.log(produit); // 240000

// Maximum
const max = nombres.reduce((acc, n) => Math.max(acc, n), -Infinity);
console.log(max); // 40

// Compter les occurrences
const votes = ['oui', 'non', 'oui', 'oui', 'non', 'abstention'];
const compteur = votes.reduce((acc, v) => {
  acc[v] = (acc[v] || 0) + 1;
  return acc;
}, {});
console.log(compteur);
// → { oui: 3, non: 2, abstention: 1 }

// Grouper par catégorie
const produits = [
  { nom: 'Clavier', categorie: 'informatique', prix: 50 },
  { nom: 'Chaise', categorie: 'mobilier', prix: 150 },
  { nom: 'Souris', categorie: 'informatique', prix: 30 }
];

const groupes = produits.reduce((acc, p) => {
  (acc[p.categorie] = acc[p.categorie] || []).push(p);
  return acc;
}, {});
console.log(groupes);
// → { informatique: [...], mobilier: [...] }

find, findIndex, some, every

Ces méthodes testent ou recherchent des éléments dans le tableau.

JavaScript — find / some / every 
const personnes = [
  { nom: 'Alice', age: 25 },
  { nom: 'Bob', age: 17 },
  { nom: 'Charlie', age: 30 }
];

// find() — premier élément qui correspond (ou undefined)
const majeur = personnes.find(p => p.age >= 18);
console.log(majeur); // { nom: "Alice", age: 25 }

// findIndex() — index du premier élément trouvé (-1 si absent)
const indexBob = personnes.findIndex(p => p.nom === 'Bob');
console.log(indexBob); // 1

// some() — au moins un élément correspond ?
const aDesMineurs = personnes.some(p => p.age < 18);
console.log(aDesMineurs); // true (Bob a 17 ans)

// every() — tous les éléments correspondent ?
const tousMajeurs = personnes.every(p => p.age >= 18);
console.log(tousMajeurs); // false

// Avec nombres
const notes = [12, 15, 8, 19, 10];
console.log(notes.some(n => n > 18)); // true (19)
console.log(notes.every(n => n >= 10)); // false (8)

// includes() — existe-t-il ?
console.log(notes.includes(15));  // true
console.log(notes.includes(20));  // false

08

sort et spread operator

sort trie le tableau sur place. Le spread ... copie ou combine des tableaux.

JavaScript — sort et spread 
// ── sort ──
const nombres = [30, 1, 100, 5, 20];

// Attention : sort trie par défaut en chaîne de caractères !
const mauvaisTri = [...nombres].sort();
console.log(mauvaisTri); // [1, 100, 20, 30, 5] ← pas ce qu'on veut

// Tri numérique croissant
const croissant = [...nombres].sort((a, b) => a - b);
console.log(croissant); // [1, 5, 20, 30, 100]

// Tri décroissant
const decroissant = [...nombres].sort((a, b) => b - a);
console.log(decroissant); // [100, 30, 20, 5, 1]

// Trier des objets
const personnes = [
  { nom: 'Alice', age: 25 },
  { nom: 'Bob', age: 17 },
  { nom: 'Charlie', age: 30 }
];
const parAge = [...personnes].sort((a, b) => a.age - b.age);
console.log(parAge);

// ── Spread operator ... ──
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// Copie (ne mute pas l'original)
const copie = [...arr1];
console.log(copie); // [1, 2, 3]
console.log(arr1 === copie); // false

// Fusion
const fusion = [...arr1, ...arr2];
console.log(fusion); // [1, 2, 3, 4, 5, 6]

// Ajout au milieu
const avecMilieu = [0, ...arr1, 10];
console.log(avecMilieu); // [0, 1, 2, 3, 10]

// Spread sur chaîne
const lettres = [...'Hello'];
console.log(lettres); // ["H", "e", "l", "l", "o"]

// Spread sur objet (copie + extension)
const base = { x: 1, y: 2 };
const etendu = { ...base, z: 3 };
console.log(etendu); // { x: 1, y: 2, z: 3 }

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Mutation : push, pop, shift, unshift, splice
  • Interrogation : length, indexOf, includes, join, concat
  • forEach : boucle fonctionnelle (pas de retour)
  • map : transformation 1→1, retourne un nouveau tableau
  • filter : sélection, retourne un sous-ensemble
  • reduce : accumulation à une valeur unique
  • find, findIndex, some, every : recherche et tests
  • sort (avec fonction comparateur) et spread operator

Mini-glossaire

Mutation
Modification directe du tableau original (push, pop, splice, sort). À manipuler avec précaution.
map
Méthode qui crée un nouveau tableau en appliquant une fonction à chaque élément.
filter
Méthode qui crée un nouveau tableau avec les éléments passant un test booléen.
reduce
Méthode qui réduit un tableau à une valeur unique via un accumulateur.
Spread (...)
Opérateur qui "déplie" un itérable en éléments individuels. Copie superficielle.
Fonction de comparaison
Fonction passée à sort : (a, b) => a - b pour tri croissant.
Chaînage (chaining)
Enchaîner plusieurs méthodes : arr.filter(...).map(...).reduce(...).
