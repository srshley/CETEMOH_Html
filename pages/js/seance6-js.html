01

Qu'est-ce que le DOM ?

🖱️ Clic utilisateur → 📡 Événement DOM
📡 Événement DOM → ⚡ addEventListener
⚡ addEventListener → 🎯 Fonction callback
🎯 Fonction callback → ✨ DOM mis à jour

Le DOM (Document Object Model) est une représentation de la page HTML sous forme d'arbre d'objets. JavaScript peut lire, modifier, ajouter ou supprimer des éléments via le DOM.

💡
DOM ≠ HTML source

Le DOM est l'interface entre JS et le HTML affiché. Quand vous modifiez le DOM, la page se met à jour instantanément, mais le fichier .html original n'est pas modifié.

JavaScript — Hiérarchie DOM 
// document — point d'entrée du DOM
console.log(document);          // #document
console.log(document.head);     // <head>
console.log(document.body);     // <body>
console.log(document.title);    // titre de l'onglet
console.log(document.URL);      // URL de la page

// Quelques propriétés utiles
console.log(document.documentElement); // <html>
console.log(document.forms);    // collection des formulaires
console.log(document.images);   // collection des images
console.log(document.links);    // collection des liens
02

querySelector — sélectionner un élément

querySelector retourne le premier élément correspondant au sélecteur CSS.

JavaScript 
// Par balise
const titre = document.querySelector('h1');

// Par classe
const info = document.querySelector('.info');

// Par id
const header = document.querySelector('#header');

// Sélecteur CSS complexe
const premierLi = document.querySelector('ul li:first-child');
const inputEmail = document.querySelector('input[type="email"]');

// Parent → enfant
const contenu = document.querySelector('.contenu p');

// Si l'élément n'existe pas → null
const inexistant = document.querySelector('.rien');
console.log(inexistant); // null

03

querySelectorAll — sélection multiple

querySelectorAll retourne une NodeList (statique) de tous les éléments correspondants.

JavaScript 
// Tous les paragraphes
const tousLesP = document.querySelectorAll('p');
console.log(tousLesP.length); // nombre de paragraphes

// Tous les éléments avec la classe .item
const items = document.querySelectorAll('.item');

// Sélecteurs multiples
const titres = document.querySelectorAll('h1, h2, h3');

// Avec attribut
const actifs = document.querySelectorAll('[data-actif="true"]');

// Parcourir une NodeList
items.forEach(item => {
  console.log(item.textContent);
});

// Convertir en vrai tableau avec spread
const tableau = [...items];
tableau.map(item => item.textContent);
📌
NodeList vs HTMLCollection

querySelectorAll renvoie une NodeList (statique, supporte forEach). Les anciennes méthodes (getElementsByTagName, getElementsByClassName) renvoient une HTMLCollection (live, pas de forEach). Préférez querySelectorAll.

04

textContent et innerHTML

textContent lit/modifie le texte seul. innerHTML lit/modifie le HTML complet.

JavaScript 
const zone = document.getElementById('demo-zone');

// Lire le texte (ignorer le HTML)
console.log(zone.textContent);

// Modifier le texte (toute balise est échappée)
zone.textContent = 'Texte simple';
// Affiche : Texte <strong>simple</strong>

// Lire le HTML
console.log(zone.innerHTML);

// Modifier le HTML
zone.innerHTML = 'Texte gras';
// Affiche : Texte en gras

// innerHTML est pratique mais risqué (XSS) !
// Ne JAMAIS mettre de contenu utilisateur dans innerHTML

05

createElement & appendChild

Créer dynamiquement des éléments dans le DOM.

JavaScript 
// 1. Créer l'élément
const nouveauP = document.createElement('p');

// 2. Ajouter du contenu
nouveauP.textContent = 'Ce paragraphe a été créé par JS !';
nouveauP.style.color = '#3B82F6';
nouveauP.style.fontWeight = 'bold';

// 3. Insérer dans le DOM
const container = document.getElementById('demo-container');
container.appendChild(nouveauP);

// Ajouter plusieurs éléments
const liste = document.createElement('ul');
for (let i = 1; i <= 3; i++) {
  const li = document.createElement('li');
  li.textContent = `Item ${i}`;
  liste.appendChild(li);
}
container.appendChild(liste);

// insertBefore et after/before (modernes)
const span = document.createElement('span');
span.textContent = ' ★ ';
container.firstChild?.before(span);  // avant le premier enfant
container.prepend(span);  // ou au début
container.append('fin');  // texte directement
RENDU

06

classList — gérer les classes CSS

classList permet d'ajouter, retirer, basculer ou vérifier les classes CSS.

JavaScript 
const demo = document.getElementById('demo-class');

// add() — ajouter une classe
demo.classList.add('actif');

// remove() — retirer une classe
demo.classList.remove('inactif');

// toggle() — ajoute si absente, retire si présente
demo.classList.toggle('survole');

// contains() — vérifier si la classe existe
if (demo.classList.contains('actif')) {
  console.log('La classe actif est présente');
}

// replace() — remplacer une classe
demo.classList.replace('ancien', 'nouveau');

// Multi-classes
demo.classList.add('rouge', 'gras', 'bordure');

// Style conditionnel
const estActif = demo.classList.contains('actif');
demo.style.background = estActif ? '#e0f7e0' : '#f0f0f0';

07

Propriété style — CSS inline

La propriété style permet de modifier le CSS inline d'un élément.

JavaScript 
const boite = document.getElementById('demo-style');

// Propriétés individuelles (camelCase)
boite.style.backgroundColor = '#3B82F6';
boite.style.color = 'white';
boite.style.padding = '1rem';
boite.style.borderRadius = '8px';
boite.style.fontSize = '1.2rem';
boite.style.fontWeight = 'bold';
boite.style.textAlign = 'center';

// Propriétés avec préfixe -webkit-
boite.style.WebkitTransition = 'all 0.3s';

// Lire le style calculé (CSS réellement appliqué)
const styles = getComputedStyle(boite);
console.log(styles.fontSize); // "1.2rem"

// cssText — tout écrire d'un coup
boite.style.cssText = 'background:#28a745;color:#fff;padding:1rem;border-radius:8px;';

08

Dataset — attributs data-*

Les attributs data-* stockent des données personnalisées dans le HTML. Accessibles via element.dataset.

JavaScript 
// En HTML : <div data-id="42" data-nom="Alice" data-role="admin">

const user = document.getElementById('demo-user');

// Lire les data-*
console.log(user.dataset.id);     // "42"
console.log(user.dataset.nom);    // "Alice"
console.log(user.dataset.role);   // "admin"

// Modifier
user.dataset.role = 'utilisateur';

// Ajouter une nouvelle data-*
user.dataset.dateInscription = '2024-06-15';

// Vérifier
console.log(user.dataset.role); // "utilisateur"
console.log(user.dataset.dateInscription); // "2024-06-15"

// Data-* avec kebab-case → dataset en camelCase
// data-user-name → dataset.userName
const utilisateur = document.getElementById('demo-user2');
console.log(utilisateur.dataset.userName); // "Bob"

// Supprimer un data-*
delete user.dataset.temp;

// Parcourir tous les data-*
for (const [key, value] of Object.entries(user.dataset)) {
  console.log(`${key}: ${value}`);
}

Exercices pratiques

EX 01 🏗️ Construction dynamique
  1. Créez une page HTML avec un <div id="app"></div>.
  2. En JS, créez un <h1> avec "Mon site dynamique" et insérez-le dans #app.
  3. Créez une liste <ul> avec 4 éléments <li>.
  4. Ajoutez un paragraphe avec textContent.
  5. Modifiez la couleur du titre avec style.
  6. Ajoutez une classe "actif" au premier <li> avec classList.
EX 02 📋 Galerie de données
  1. Créez un tableau const fruits = [{nom:"Pomme", couleur:"rouge"}, {nom:"Banane", couleur:"jaune"}, {nom:"Kiwi", couleur:"vert"}].
  2. Pour chaque fruit, créez une <div> avec le nom et la couleur en style.
  3. Ajoutez un data-prix aléatoire à chaque fruit affiché.
  4. Utilisez querySelectorAll pour sélectionner toutes les divs créées.
  5. Au clic sur chaque div, affichez son data-prix dans la console.
  6. Créez un bouton "Tout supprimer" qui vide le conteneur.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Concept du DOM : arbre d'objets, point d'entrée document
  • querySelector (1 élément) et querySelectorAll (NodeList)
  • textContent (texte brut) vs innerHTML (HTML interprété)
  • createElement, appendChild, insertBefore, prepend, after
  • classList : add, remove, toggle, contains, replace
  • Propriété style et getComputedStyle
  • Attributs data-* et l'objet dataset

Mini-glossaire

DOM
Document Object Model — interface de programmation qui représente la page HTML sous forme d'arbre d'objets.
NodeList
Collection d'éléments retournée par querySelectorAll. Statique, itérable avec forEach.
textContent
Propriété pour lire/modifier le texte sans balises HTML. Plus performant et plus sûr que innerHTML.
innerHTML
Propriété pour lire/modifier le HTML interne. Risque XSS si le contenu vient de l'utilisateur.
classList
Objet avec les méthodes add/remove/toggle/contains pour manipuler les classes CSS.
dataset
Objet donnant accès aux attributs data-* d'un élément (data-nom → dataset.nom).
