01

Tableau basique : <table>, <tr>, <td>

Un tableau HTML se construit ligne par ligne. <table> est le conteneur, <tr> (table row) chaque ligne, <td> (table data) chaque cellule.

HTML — Table basique 
<table>
  <tr>
    <td>Ligne 1, Cellule 1</td>
    <td>Ligne 1, Cellule 2</td>
  </tr>
  <tr>
    <td>Ligne 2, Cellule 1</td>
    <td>Ligne 2, Cellule 2</td>
  </tr>
  <tr>
    <td>Ligne 3, Cellule 1</td>
    <td>Ligne 3, Cellule 2</td>
  </tr>
</table>
RENDU
Ligne 1, Cellule 1Ligne 1, Cellule 2
Ligne 2, Cellule 1Ligne 2, Cellule 2
Ligne 3, Cellule 1Ligne 3, Cellule 2
02

Thead, Tbody, Tfoot

Ces trois éléments structurent le tableau en en-tête, corps et pied. Ils améliorent la sémantique, permettent la répétition de l'en-tête à l'impression, et aident les lecteurs d'écran.

HTML — thead, tbody, tfoot 
<table>
  <thead>
    <tr>
      <th>Produit</th>
      <th>Quantité</th>
      <th>Prix</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Pommes</td>
      <td>3</td>
      <td>2.50 €</td>
    </tr>
    <tr>
      <td>Bananes</td>
      <td>6</td>
      <td>1.80 €</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">Total</td>
      <td>4.30 €</td>
    </tr>
  </tfoot>
</table>
RENDU
ProduitQuantitéPrix
Pommes32.50 €
Bananes61.80 €
Total4.30 €
03

Th & scope

<th> (table header) remplace <td> pour les cellules d'en-tête. L'attribut scope indique si l'en-tête s'applique à une colonne (col) ou une ligne (row). Essentiel pour l'accessibilité.

HTML — th et scope 
<table>
  <thead>
    <tr>
      <th scope="col">Nom</th>
      <th scope="col">Âge</th>
      <th scope="col">Ville</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Jean</th>
      <td>28</td>
      <td>Paris</td>
    </tr>
    <tr>
      <th scope="row">Marie</th>
      <td>32</td>
      <td>Lyon</td>
    </tr>
  </tbody>
</table>
RENDU
NomÂgeVille
Jean28Paris
Marie32Lyon
04

Colspan — fusion de colonnes

colspan="N" étend une cellule sur N colonnes. Utile pour les titres de groupes, les totaux, ou les entêtes communes.

HTML — Colspan 
<table>
  <tr>
    <th colspan="2">Coordonnées</th>
  </tr>
  <tr>
    <td>Email</td>
    <td>contact@exemple.com</td>
  </tr>
  <tr>
    <td>Téléphone</td>
    <td>+33 1 23 45 67 89</td>
  </tr>
</table>
RENDU
Coordonnées
Emailcontact@exemple.com
Téléphone+33 1 23 45 67 89
05

Rowspan — fusion de lignes

rowspan="N" étend une cellule sur N lignes. Les cellules des lignes "absorbées" ne doivent plus apparaître dans le code.

HTML — Rowspan 
<table>
  <tr>
    <th rowspan="2">Période</th>
    <th>Matin</th>
    <th>Après-midi</th>
  </tr>
  <tr>
    <td>HTML</td>
    <td>CSS</td>
  </tr>
  <tr>
    <th rowspan="2">Saison 2</th>
    <td>JavaScript</td>
    <td>PHP</td>
  </tr>
  <tr>
    <td>Bases de données</td>
    <td>Frameworks</td>
  </tr>
</table>
RENDU
PériodeMatinAprès-midi
HTMLCSS
Saison 2JavaScriptPHP

Caption & résumé

<caption> donne un titre au tableau, visible par tous. Il doit être le premier enfant de <table>. Pour un résumé plus long (invisible), on utilise summary (déprécié en HTML5 — préférez un paragraphe avec aria-describedby).

HTML — Caption 
<table>
  <caption>Ventes du premier trimestre 2024</caption>
  <thead>
    <tr>
      <th scope="col">Mois</th>
      <th scope="col">CA HT</th>
      <th scope="col">TVA</th>
      <th scope="col">CA TTC</th>
    </tr>

      <td>12 000 €</td>

Styler les tableaux avec CSS

Le tableau brut n'a pas de bordure visible. On utilise CSS pour ajouter des bordures, des couleurs alternées, et améliorer la lisibilité. Les classes CSS prédéfinies .table-wrap et .table-demo sont disponibles.

HTML — Table stylée 
<table>
  <caption>Horaires d'ouverture</caption>
  <thead>
    <tr>
      <th scope="col">Jour</th>
      <th scope="col">Matin</th>
      <th scope="col">Après-midi</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Lundi</th>
      <td>9h - 12h</td>
      <td>14h - 18h</td>
    </tr>
    <tr>
      <th scope="row">Mardi</th>
      <td>9h - 12h</td>
      <td>14h - 18h</td>
    </tr>
    <tr>
      <th scope="row">Mercredi</th>
      <td>9h - 12h</td>
      <td>14h - 18h</td>
    </tr>
    <tr>
      <th scope="row">Jeudi</th>
      <td>9h - 12h</td>
      <td>14h - 18h</td>
    </tr>
    <tr>
      <th scope="row">Vendredi</th>
      <td>9h - 12h</td>
      <td>14h - 17h</td>
    </tr>
  </tbody>
</table>

08

Accessibilité des tableaux

Un tableau accessible repose sur trois piliers : scope sur les th, un caption ou un résumé textuel, et une structure logique avec thead/tbody/tfoot.

💡
Lecteurs d'écran et tableaux

Les lecteurs d'écran annoncent "Tableau de 3 colonnes et 5 lignes" puis lisent les en-têtes à chaque changement de cellule. Sans scope, l'utilisateur perd le contexte. Sans caption, il ne sait pas ce que représente le tableau.

⚠️
N'utilisez pas les tableaux pour la mise en page

Les tableaux sont faits pour des données tabulaires (horaires, prix, statistiques). Pas pour créer des colonnes ou des layouts. Utilisez CSS Grid ou Flexbox pour la mise en page.

✅ Usage correct

Horaires, fiches de paie, classements, calendriers, inventaires, données financières.

❌ Usage incorrect

Mise en page en colonnes, barre de navigation, galerie d'images, formulaire.

09

Exercices pratiques

EX 01 ✏️ Emploi du temps
  1. Créez un tableau représentant votre emploi du temps.
  2. Colonnes : Lundi au Vendredi. Lignes : 8h-10h, 10h-12h, 13h-15h, 15h-17h.
  3. Utilisez <thead>, <tbody>, <th scope="col"> et <th scope="row">.
  4. Ajoutez un <caption> "Emploi du temps Semaine".
  5. Fusionnez les cellules de pause déjeuner avec rowspan.
EX 02 ✏️ Facture
  1. Créez un tableau de facture avec : Produit, Qté, Prix unitaire, Total.
  2. Ajoutez 5 produits. Utilisez colspan pour la ligne de total.
  3. Ajoutez <tfoot> avec le total général.
EX 03 🔍 Classement
  1. Créez un classement de 5 sports ou jeux vidéo.
  2. Colonnes : Rang, Nom, Score, Durée.
  3. Utilisez scope correctement sur tous les <th>.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Tableau basique : table, tr, td
  • Structuration sémantique : thead, tbody, tfoot
  • Cellules d'en-tête th et attribut scope
  • Fusion de colonnes : colspan
  • Fusion de lignes : rowspan
  • Caption : titre visible du tableau
  • Styler les tableaux : bordures, alternance de couleurs
  • Accessibilité : scope, caption, thead/tbody/tfoot

Mini-glossaire

table
Conteneur principal du tableau.
tr
Table row — une ligne du tableau.
td
Table data — cellule de donnée.
th
Table header — cellule d'en-tête. Gras et centré par défaut.
scope
Indique la portée de l'en-tête : "col" (colonne) ou "row" (ligne).
colspan
Fusionne N colonnes horizontalement.
rowspan
Fusionne N lignes verticalement.
caption
Titre du tableau. Premier enfant de <table>.
thead
En-tête du tableau. Contient les th.
tbody
Corps du tableau avec les td.
tfoot
Pied du tableau. Souvent utilisé pour les totaux.
