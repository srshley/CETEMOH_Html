01

La balise <img>

<img> est une balise orpheline qui insère une image dans le flux du document. Trois attributs sont essentiels : src (source), alt (texte alternatif), et les dimensions width/height.

🖼️photo.jpg600×400 px
alt="Chat roux endormi sur canapé"
src ★
Source — chemin local ou URL.
alt ★
Texte alternatif — lu par lecteurs d'écran, indexé par Google.
width/height
Dimensions en px. Évite le CLS (saut de mise en page).
loading="lazy"
Charge l'image seulement quand visible.
HTML — Image basique 
<img src="images/chat.jpg"
     alt="Un chat roux endormi sur un canapé bleu"
     width="600" height="400"
     loading="lazy"
     decoding="async">
RENDU
📌
Guide de l'attribut alt

✅ alt="Portrait de Marie Curie en 1903"
✅ alt="" (image décorative, lecteur d'écran l'ignore)
❌ alt="image" ou alt="DSC_4892.jpg"

02

Figure & Figcaption

<figure> associe une image (ou un autre média) à sa légende <figcaption>. L'ensemble forme un bloc sémantique unique, déplaçable sans perdre le contexte.

HTML — Figure + Figcaption 
<figure>
  <img src="tokyo.jpg"
       alt="Vue aérienne de Tokyo au coucher du soleil"
       width="800" height="500"
       loading="lazy">
  <figcaption>Tokyo vu depuis la Tokyo Skytree,
    juillet 2024. Photo : Jean Dupont.
  </figcaption>
</figure>
RENDU

Formats d'image

Chaque format a ses forces et faiblesses. Choisir le bon format améliore la performance et la qualité visuelle.

JPEG
Photographies
✓ Très compressé
✓ Universel
✗ Pas de transparence
PNG
Logos & icônes
✓ Transparence
✓ Sans perte
✗ Fichier lourd
WebP
Polyvalent
✓ 30% + léger que JPEG
✓ Transparence
✓ Animations

Compression max
✓ 50% + léger que JPEG
✓ Qualité excellente
✗ Support partiel
SVG
Vectoriel
✓ Infiniment redimensionnable
✓ Très léger
✓ Animable
GIF
Animations
✓ Animations
✓ Universel
✗ 256 couleurs max
04

Srcset & sizes

srcset permet de fournir plusieurs versions d'une même image. Le navigateur choisit la plus adaptée à la résolution et à la taille de l'écran. sizes indique la largeur d'affichage prévue.

HTML — Srcset et sizes 
<img src="image-800.jpg"
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1600.jpg 1600w"
     sizes="(max-width: 480px) 400px,
            (max-width: 1024px) 800px,
            1600px"
     alt="Paysage de montagne"
     width="1600" height="900"
     loading="lazy">
RENDU

Le navigateur choisit automatiquement la meilleure version selon l'écran. Sur mobile : 400w. Sur desktop : 1600w.

05

Balise <picture>

<picture> offre un contrôle total : choisir le format selon le support du navigateur (WebP, AVIF) ET l'art direction (image différente selon la largeur).

HTML — Picture 
<picture>
  <!-- AVIF : meilleure compression -->
  <source srcset="photo.avif" type="image/avif">

  <!-- WebP : bon compromis -->
  <source srcset="photo.webp" type="image/webp">

  <!-- Fallback JPEG universel -->
  <img src="photo.jpg"
       alt="Coucher de soleil sur la mer"
       width="800" height="533"
       loading="lazy">
</picture>
RENDU

Si le navigateur supporte AVIF → utilise photo.avif. Sinon WebP, sinon JPEG en fallback. L'image s'affiche toujours.

06

Vidéo HTML5

La balise <video> intègre une vidéo sans plugin. Attributs clés : src, controls, autoplay (avec muted obligatoire), loop, poster.

HTML — Vidéo 
<!-- Vidéo basique avec contrôles -->
<video src="presentation.mp4"
       controls
       width="640" height="360"
       poster="miniature.jpg">
  <p>Votre navigateur ne supporte pas la vidéo.</p>
</video>

<!-- Vidéo avec sources multiples (fallback) -->
<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <p>Vidéo non supportée.</p>
</video>

<!-- Autoplay muet (autorisé par les navigateurs) -->
<video src="animation.mp4"
       autoplay muted loop
       width="400" height="300"></video>
RENDU
📌
Autoplay et navigateurs modernes

Depuis 2018, Chrome, Firefox et Safari bloquent l'autoplay avec son. Pour qu'une vidéo se lance automatiquement, ajoutez muted (muet). L'utilisateur peut ensuite réactiver le son.

07

Audio HTML5

La balise <audio> fonctionne comme <video> mais pour le son. Attributs : src, controls, autoplay, loop, muted.

HTML — Audio 
<!-- Audio basique avec contrôles -->
<audio src="musique.mp3" controls>
  <p>Votre navigateur ne supporte pas l'audio.</p>
</audio>

<!-- Sources multiples (fallback) -->
<audio controls>
  <source src="musique.mp3" type="audio/mpeg">
  <source src="musique.ogg" type="audio/ogg">
  <p>Audio non supporté.</p>
</audio>

<!-- Boucle et autoplay muet -->
<audio src="ambiance.mp3"
       autoplay loop muted></audio>
RENDU
08

Iframe — Intégrer YouTube

<iframe> intègre une page web dans une autre. Utilisé pour les vidéos YouTube, les cartes Google Maps, les widgets. Attention aux dimensions et à l'attribut title pour l'accessibilité.

HTML — Iframe YouTube 
<!-- Intégration YouTube (cliquer sur Partager → Intégrer) -->
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="Vidéo tutoriel HTML"
  frameborder="0"
  allow="accelerometer; autoplay;
         clipboard-write; encrypted-media;
         gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>

<!-- Google Maps -->
<iframe
  src="https://www.google.com/maps/embed?pb=..."
  width="600" height="450"
  style="border:0;"
  allowfullscreen=""
  loading="lazy"
  title="Carte Google Maps">
</iframe>
RENDU
▶️

Vidéo YouTube intégrée ici

💡
Obtenir le code d'intégration YouTube

Sur YouTube : cliquez sur "Partager" sous la vidéo → "Intégrer" → copiez le code <iframe>. Remplacez VIDEO_ID par l'ID de la vidéo (ex: dQw4w9WgXcQ).

09

Exercices pratiques

EX 01 ✏️ Galerie d'images
  1. Créez une page "Galerie" avec 4 images locales (utilisez celles du dossier images/).
  2. Chaque image a alt descriptif, width, height et loading="lazy".
  3. Encadrez chaque image dans <figure> avec <figcaption>.
  4. Rendez la première image cliquable (lien vers une page externe).
EX 02 🔧 Média player
  1. Ajoutez une vidéo avec des sources MP4 et WebM (fallback).
  2. Ajoutez un fichier audio avec contrôles.
  3. La vidéo a un poster (image de prévisualisation).
EX 03 🔍 Intégration YouTube
  1. Trouvez une vidéo YouTube sur le HTML ou le CSS.
  2. Cliquez "Partager" → "Intégrer" et copiez le code.
  3. Collez-le dans une page HTML.
  4. Ajoutez title et loading="lazy" à l'iframe.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • Image img : src, alt, width, height, loading lazy
  • Figure et figcaption pour les légendes sémantiques
  • Formats : JPEG, PNG, WebP, AVIF, SVG, GIF
  • Srcset et sizes pour les images responsives
  • Picture pour le choix de format et l'art direction
  • Vidéo : src, controls, autoplay, muted, loop, poster
  • Audio : src, controls, sources multiples
  • Iframe pour YouTube, Maps et autres intégrations

Mini-glossaire

alt
Texte alternatif d'une image. Obligatoire. Décrit l'image pour les lecteurs d'écran et Google.
CLS
Cumulative Layout Shift — déplacement de la mise en page au chargement des images. Évité par width+height.
loading="lazy"
Charge l'image seulement quand elle entre dans la fenêtre visible.
srcset
Liste de plusieurs résolutions d'une image pour différentes tailles d'écran.
picture
Conteneur pour sources multiples avec fallback. Utile pour les formats modernes.
figure
Contenu autonome (image, vidéo, code) avec légende optionnelle.
figcaption
Légende d'un élément figure. Toujours à l'intérieur de figure.
iframe
Fenêtre intégrée affichant une autre page web. Utilisé pour YouTube, Maps.
controls
Attribut booléen qui affiche les boutons de contrôle du média (play, pause, volume).
poster
Image de prévisualisation affichée avant le lancement d'une vidéo.
