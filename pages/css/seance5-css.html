Backgrounds & Dégradés

Couleurs de fond · Images · Répétition · Taille · Dégradés · Ombres

01

background-color

La propriété la plus simple pour colorer le fond d'un élément. Elle s'applique à la zone content + padding (pas à la margin).

🎨 Sélecteur → 📐 Propriété
📐 Propriété → 🎯 Valeur
🎯 Valeur → ✨ Résultat stylisé
CSS 
/* Couleurs unies */
.bg-red    { background-color: #e74c3c; }
.bg-blue   { background-color: #3498db; }
.bg-green  { background-color: #27ae60; }

/* Transparence */
.bg-alpha  { background-color: rgba(231, 76, 60, .3); }
.bg-hexa   { background-color: #e74c3c4d; } /* même effet avec hex alpha */

/* Transparent (défaut) */
.bg-none   { background-color: transparent; }

/* Couleur de fond du body */
body       { background-color: #f8f9fa; }
RENDU
#e74c3c #3498db #27ae60 rgba 30%
02

background-image

Ajoute une image en arrière-plan. L'image se répète par défaut pour remplir toute la boîte. Combinée avec les autres propriétés background, elle offre un contrôle précis.

CSS 
/* Image simple */
.bg-img {
  background-image: url('images/fond.jpg');
  width: 400px;
  height: 300px;
}

/* Chemin absolu relatif à la racine */
.bg-absolue { background-image: url('/images/fond.jpg'); }

/* URL externe */
.bg-externe { background-image: url('../../images/montagne.jpg'); }

/* Plusieurs backgrounds (séparés par des virgules) */
.bg-multiples {
  background-image:
    url('images/overlay.png'),
    url('images/fond.jpg');
  /* overlay.png sera au-dessus de fond.jpg */
}

/* Background shorthand */
.bg-short {
  background: #2c3e50 url('images/pattern.png') no-repeat center / cover;
}
RENDU
Image de fond
📌
background-image ne remplace pas <img>

Une image de fond (background-image) n'est pas du contenu. Elle est ignorée par les lecteurs d'écran (accessibilité) et par Google Images (SEO). Utilisez background-image pour les images décoratives et <img> pour les images qui apportent du sens.

03

background-repeat

Par défaut, une image de fond se répète en mosaïque (repeat). On peut contrôler la répétition sur chaque axe ou la supprimer.

CSS 
/* Valeurs de background-repeat */
.repeat     { background-repeat: repeat; }      /* défaut — répète sur X et Y */
.repeat-x   { background-repeat: repeat-x; }     /* répète uniquement horizontalement */
.repeat-y   { background-repeat: repeat-y; }     /* répète uniquement verticalement */
.no-repeat  { background-repeat: no-repeat; }    /* une seule occurrence */
.space      { background-repeat: space; }        /* répète avec espacement égal */
.round      { background-repeat: round; }        /* répète en ajustant la taille */

/* Cas typique — motif décoratif horizontal */
.separateur {
  background-image: url('ligne-deco.svg');
  background-repeat: repeat-x;
  height: 20px;
}
RENDU
Motif répété (repeat)
04

background-size

Contrôle les dimensions de l'image de fond. Essentiel pour les images responsives et les hero sections.

CSS 
/* Valeurs les plus utilisées */
.cover  { background-size: cover; }   /* remplit tout le conteneur, rogne si nécessaire */
.contain { background-size: contain; } /* l'image entière visible, peut laisser des espaces */

/* Dimensions explicites */
.px     { background-size: 200px 150px; }  /* largeur, hauteur */
.auto   { background-size: auto 100%; }    /* hauteur 100%, largeur automatique */
.percent { background-size: 50% auto; }    /* moitié de la largeur du conteneur */

/* Mot-clé pour les deux axes */
.bg-size { background-size: cover; background-repeat: no-repeat; }

/* Le trio gagnant pour un arrière-plan plein écran */
.hero {
  background-image: url('hero.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}
RENDU
cover
contain
💡
cover vs contain

background-size: cover : l'image remplit tout le conteneur, mais elle peut être rognée sur les bords. Idéal pour les héros et les fonds pleine largeur. background-size: contain : l'image entière est visible, mais il peut y avoir des bandes vides (letterboxing). Utile pour les logos ou les icônes en fond. Choisissez selon que vous préférez ne pas rogner (contain) ou ne pas laisser de vide (cover).

05

background-position

Positionne l'image de fond dans la boîte. Utile quand l'image est plus grande que le conteneur ou qu'elle n'est pas répétée.

CSS 
/* Mots-clés */
.center  { background-position: center; }        /* centré (X et Y) */
.top     { background-position: top center; }    /* haut centré */
.bottom  { background-position: bottom center; } /* bas centré */
.left    { background-position: center left; }   /* milieu à gauche */
.right   { background-position: right center; }  /* milieu à droite */
.corner  { background-position: top right; }     /* coin supérieur droit */

/* Valeurs en pourcentage (0% = début, 100% = fin) */
.bg-pos  { background-position: 25% 75%; } /* horizontal à 25%, vertical à 75% */

/* Valeurs en pixels */
.bg-px   { background-position: 20px 40px; } /* décalé de 20px de la gauche, 40px du haut */

/* Axes séparés */
.bg-edge { background-position: right 20px bottom 10px; } /* à 20px du bord droit, 10px du bas */
RENDU
background-position: center
06

Dégradés linéaires (linear-gradient)

Les dégradés CSS permettent de créer des transitions fluides entre plusieurs couleurs — sans image. Ils sont considérés comme des images (background-image) et supportent donc toutes les propriétés background.

CSS 
/* Direction par défaut : haut → bas */
.grad-basic {
  background: linear-gradient(#e74c3c, #3498db);
}

/* Direction explicite (to + côté) */
.grad-right { background: linear-gradient(to right, #e74c3c, #3498db); }
.grad-diag  { background: linear-gradient(to bottom right, #e74c3c, #3498db); }

/* Angle en degrés */
.grad-angle { background: linear-gradient(45deg, #e74c3c, #3498db); }

/* Plusieurs couleurs */
.grad-multi { background: linear-gradient(to right, #e74c3c, #f39c12, #27ae60, #3498db); }

/* Avec points d'arrêt (color-stop) */
.grad-stop  { background: linear-gradient(to right, #e74c3c 0%, #f39c12 50%, #27ae60 100%); }

/* Dégradé avec transparence */
.grad-alpha { background: linear-gradient(to bottom, rgba(0,0,0,.8), rgba(0,0,0,0)); }
RENDU
💡
Dégradé = image de fond

Un dégradé étant une image CSS (et non une couleur unie), il se combine avec les autres propriétés background : background-repeat, background-size, background-position. Vous pouvez aussi superposer un dégradé avec une vraie image : background: linear-gradient(rgba(0,0,0,.5), rgba(0,0,0,.5)), url('photo.jpg') center / cover;.

07

Dégradés radiaux (radial-gradient)

Les dégradés radiaux partent d'un point central (ou décentré) et s'étendent en cercles ou ellipses concentriques.

CSS 
/* Cercle parfait */
.radial-circle {
  background: radial-gradient(circle, #ff6b6b, #c0392b);
}

/* Ellipse (défaut) */
.radial-ellipse {
  background: radial-gradient(#3498db, #2c3e50);
}

/* Position décentrée */
.radial-pos {
  background: radial-gradient(circle at top left, #f39c12, #e74c3c);
}

/* Plusieurs couleurs */
.radial-multi {
  background: radial-gradient(circle, #ff6b6b 0%, #feca57 50%, #48dbfb 100%);
}

/* Taille : closest-side, farthest-side, closest-corner, farthest-corner */
.radial-size {
  background: radial-gradient(circle farthest-corner, #ff6b6b, #c0392b);
}
RENDU
08

box-shadow — ombres portées

box-shadow ajoute une ou plusieurs ombres autour de la boîte d'un élément. C'est un effet subtil qui apporte de la profondeur et du relief.

CSS 
/* Syntaxe : offset-x offset-y blur-radius spread-radius color */

/* Ombre simple — décalée vers le bas */
.shadow-1 { box-shadow: 0 2px 4px rgba(0,0,0,.1); }

/* Ombre plus prononcée */
.shadow-2 { box-shadow: 0 8px 24px rgba(0,0,0,.12); }

/* Ombre intérieure (inset) */
.shadow-inset { box-shadow: inset 0 2px 4px rgba(0,0,0,.1); }

/* Pas de flou — bordure dure */
.shadow-hard { box-shadow: 4px 4px 0 #2c3e50; }

/* Ombres multiples (séparées par des virgules) */
.shadow-multi {
  box-shadow:
    0 2px 4px rgba(0,0,0,.1),
    0 8px 16px rgba(0,0,0,.08);
}

/* Ombre colorée */
.shadow-color { box-shadow: 0 4px 12px rgba(231, 76, 60, .3); }
RENDU
Légère
Profonde
Inset
Dure
📌
Propriétés de l'ombre détaillées

offset-x (horizontal) : positif = vers la droite, négatif = vers la gauche. offset-y (vertical) : positif = vers le bas, négatif = vers le haut. blur-radius : flou de l'ombre (0 = bord dur). spread-radius : étalement (positif = agrandit l'ombre, négatif = la rétrécit). color : couleur avec transparence recommandée.

CSS — Ombres décoratives 
/* Style "carte" standard */
.card {
  background: white;
  border-radius: 8px;
  box-shadow: 0 1px 3px rgba(0,0,0,.08), 0 4px 12px rgba(0,0,0,.04);
  transition: box-shadow .2s;
}
.card:hover {
  box-shadow: 0 4px 12px rgba(0,0,0,.12), 0 12px 32px rgba(0,0,0,.08);
}

/* Bouton avec relief */
.btn {
  background: #3498db;
  color: white;
  box-shadow: 0 4px 0 #2980b9;
  /* donne un effet 3D "enfoncé" au clic */
}
.btn:active {
  box-shadow: 0 1px 0 #2980b9;
  transform: translateY(3px);
}

/* Glow (lueur) */
.glow {
  box-shadow: 0 0 20px rgba(52, 152, 219, .4);
}

Exercices pratiques

EX 01 🌈 Dégradés créatifs
  1. Créez exo-gradients.html avec 6 blocs carrés de 150×150px.
  2. Chaque bloc a un dégradé différent : linéaire horizontal, linéaire diagonal, linéaire avec 4 couleurs, radial circulaire, radial décentré, radial à 3 couleurs.
  3. Ajoutez un dégradé de transparence sur le dernier (rgba → transparent).
  4. Ajoutez une ombre portée à chaque bloc avec box-shadow.
  5. Créez une hero section avec un dégradé linéaire + une image par-dessus (superposition).
EX 02 🖼️ Mise en page avec backgrounds
  1. Créez exo-backgrounds.html avec une galerie de 4 cartes.
  2. Chaque carte a une image de fond différente (utilisez les images du dossier images/) avec background-size: cover et background-position: center.
  3. Ajoutez un overlay semi-transparent (dégradé noir → transparent) sur chaque carte pour que le texte soit lisible.
  4. Chaque carte a un titre et un sous-titre en blanc.
  5. Utilisez background-size: contain pour un bloc avec un logo/icône.
  6. Ajoutez un motif répété avec background-repeat pour un bloc décoratif.
🔍
Exercice bonus — Système d'ombres cohérent

Créez un système de 4 niveaux d'ombre : shadow-sm (subtile), shadow-md (moyenne), shadow-lg (grande), shadow-xl (très grande). Appliquez-les à des cartes. Pour chaque niveau, multipliez approximativement les valeurs par 2. Exemple : sm = 0 1px 3px, md = 0 4px 12px, lg = 0 8px 24px, xl = 0 16px 48px. Observez comment la profondeur perçue augmente.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • background-color : couleur unie, transparent, alpha
  • background-image : url(), plusieurs images, superposition
  • background-repeat : repeat, no-repeat, repeat-x, repeat-y, space, round
  • background-size : cover, contain, dimensions explicites
  • background-position : mots-clés, pourcentages, pixels
  • Linear-gradient : directions (to, angles), couleurs, color-stops, transparence
  • Radial-gradient : cercle, ellipse, position, formes
  • box-shadow : offset, blur, spread, couleur, multiples ombres, inset

Mini-glossaire

background-image
Image de fond. Accepte url(), dégradés, ou plusieurs valeurs séparées par des virgules.
background-size: cover
L'image de fond remplit tout le conteneur, rognée si nécessaire pour couvrir.
background-size: contain
L'image entière est visible, peut laisser des bandes vides.
linear-gradient
Dégradé en ligne droite. Direction : to right, 45deg, to bottom.
radial-gradient
Dégradé radial en cercle ou ellipse. Position : circle at top left.
color-stop
Point d'arrêt d'une couleur dans un dégradé. Ex : red 50%.
box-shadow
Ombre portée autour d'un élément. Syntaxe : offset-x offset-y blur spread color.
inset
Mot-clé optionnel de box-shadow qui place l'ombre à l'intérieur de la boîte.
overlay
Superposition semi-transparente (dégradé ou couleur) pour améliorer la lisibilité du texte.
shorthand background
Propriété raccourcie : background: color image repeat attachment position / size.
