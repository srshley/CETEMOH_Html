01

display: flex — le conteneur flexible

Flexbox est un modèle de mise en page unidimensionnel (une ligne ou une colonne). En déclarant display: flex sur un conteneur, ses enfants deviennent des éléments flexibles qui s'alignent automatiquement.

🎨 Sélecteur → 📐 Propriété
📐 Propriété → 🎯 Valeur
🎯 Valeur → ✨ Résultat stylisé
CSS 
/* Le conteneur flex — une simple déclaration suffit */
.container {
  display: flex;
  /* Par défaut : les enfants s'alignent horizontalement */
  /* Pas de retour à la ligne automatique */
  /* Les enfants s'étirent pour remplir la hauteur */
}

/* Les enfants deviennent flexibles */
.child {
  /* Ils peuvent être redimensionnés, déplacés, réordonnés */
}

📌
Flexbox résout les problèmes historiques

Avant Flexbox, aligner verticalement un élément était un cauchemar (padding, line-height, table-cell…). Avoir des colonnes de hauteur égale demandait des hacks CSS. Centrer parfaitement un élément dans son parent nécessitait des calculs. Flexbox résout tous ces problèmes avec des propriétés simples et intuitives.

02

flex-direction — l'axe principal

flex-direction définit la direction de l'axe principal (main axis). L'axe secondaire (cross axis) est perpendiculaire.

CSS 
/* DIRECTION PAR DÉFAUT : ligne horizontale */
.row        { flex-direction: row; }            /* de gauche à droite (défaut) */
.row-reverse{ flex-direction: row-reverse; }    /* de droite à gauche */

/* COLONNE : axe vertical */
.column         { flex-direction: column; }         /* de haut en bas */
.column-reverse { flex-direction: column-reverse; } /* de bas en haut */
RENDU
row row row
column column
03

flex-wrap & gap

flex-wrap permet aux éléments de passer à la ligne quand ils ne tiennent plus. gap ajoute un espacement uniforme entre les éléments flexibles.

CSS 
/* flex-wrap : passage à la ligne */
.nowrap  { flex-wrap: nowrap; }   /* défaut — les éléments rétrécissent pour tenir */
.wrap    { flex-wrap: wrap; }     /* passage à la ligne si nécessaire */
.wrap-rev{ flex-wrap: wrap-reverse; } /* wrap mais en ordre inverse */

/* gap : espacement entre les éléments (pas aux bords) */
.gap-8   { gap: 8px; }           /* 8px entre chaque élément */
.gap-row { row-gap: 16px; }      /* espacement vertical uniquement */
.gap-col { column-gap: 24px; }   /* espacement horizontal uniquement */

/* flex-flow = flex-direction + flex-wrap (shorthand) */
.flow    { flex-flow: row wrap; }  /* ligne horizontale avec retour à la ligne */
.flow2   { flex-flow: column nowrap; } /* colonne sans retour */
RENDU
wrap
wrap
wrap
wrap
💡
gap remplace les margins sur les enfants

Avant gap (arrivé en 2021 pour Flexbox), on devait utiliser margin sur les enfants avec des sélecteurs :first-child/:last-child pour éviter les marges aux bords. Avec gap, c'est beaucoup plus simple : une seule déclaration sur le conteneur, pas de marge aux extrémités. Supporté partout aujourd'hui (Edge 84+, Chrome 84+, Firefox 63+).

04

justify-content — alignement sur l'axe principal

justify-content distribue l'espace entre et autour des éléments flexibles sur l'axe principal (horizontal par défaut).

CSS 
/* Valeurs principales */
.flex-start    { justify-content: flex-start; }    /* groupés au début (défaut) */
.flex-end      { justify-content: flex-end; }      /* groupés à la fin */
.center        { justify-content: center; }         /* groupés au centre */
.space-between { justify-content: space-between; }  /* espace égal entre, collés aux bords */
.space-around  { justify-content: space-around; }   /* espace autour de chaque élément */
.space-evenly  { justify-content: space-evenly; }   /* espace parfaitement égal */
RENDU
space-between space-between space-between
center center
📌
Raccourci : centrer parfaitement avec Flexbox

Une des utilisations les plus fréquentes de Flexbox : display: flex; justify-content: center; align-items: center; dans un conteneur — son contenu est centré parfaitement, horizontalement ET verticalement. C'est la solution moderne à un problème qui a hanté les développeurs pendant 20 ans.

05

align-items & align-content

align-items aligne les éléments sur l'axe secondaire (perpendiculaire à l'axe principal). align-content distribue l'espace entre les lignes (quand il y a plusieurs lignes, avec flex-wrap).

CSS 
/* ALIGN-ITEMS — alignement sur l'axe secondaire */
.stretch   { align-items: stretch; }   /* étire les enfants pour remplir (défaut) */
.flex-start{align-items: flex-start; } /* alignés en haut */
.flex-end  { align-items: flex-end; }  /* alignés en bas */
.center    { align-items: center; }    /* centrés verticalement */
.baseline  { align-items: baseline; }  /* alignés sur la ligne de base du texte */

/* ALIGN-CONTENT — espacement entre les lignes (avec wrap) */
.content-center  { align-content: center; }
.content-between { align-content: space-between; }
RENDU
center center center
flex-start flex-start
💡
Récapitulatif visuel des axes Flexbox

En flex-direction: row (défaut) : main axis = horizontal (→), cross axis = vertical (↓). justify-content gère le main axis, align-items gère le cross axis. En flex-direction: column : c'est l'inverse — justify-content gère la verticale, align-items gère l'horizontale.

06

flex-grow & flex-shrink

flex-grow contrôle la capacité d'un élément à s'agrandir pour occuper l'espace disponible. flex-shrink contrôle sa capacité à rétrécir quand l'espace manque.

CSS 
/* flex-grow : proportion d'agrandissement */
.grow-0 { flex-grow: 0; }  /* ne s'agrandit pas (défaut) */
.grow-1 { flex-grow: 1; }  /* prend 1 part de l'espace libre */
.grow-2 { flex-grow: 2; }  /* prend 2 parts → 2× plus large que grow-1 */
.grow-3 { flex-grow: 3; }  /* prend 3 parts */

/* Exemple : 3 éléments, grow: 1 chacun → largeurs égales */
/* 2 éléments, grow: 1 et grow: 2 → le second est 2× plus large */

/* flex-shrink : capacité à rétrécir */
.shrink-0 { flex-shrink: 0; }  /* ne rétrécit pas (garde sa taille même si ça dépasse) */
.shrink-1 { flex-shrink: 1; }  /* peut rétrécir (défaut) */
RENDU
grow:1
grow:2
grow:1
📌
La recette des colonnes de hauteur égale

flex-grow: 1 sur tous les enfants d'un conteneur flex with align-items: stretch (valeur par défaut) donne des colonnes de hauteur égale automatiquement. C'est une des forces majeures de Flexbox. Pas besoin de hauteur fixe, pas de JavaScript, pas de hack — ça marche nativement.

07

flex-basis & le shorthand flex

flex-basis définit la taille initiale d'un élément avant que l'espace libre soit distribué (par flex-grow) ou retiré (par flex-shrink). C'est comme un width pour flex-direction: row, ou un height pour flex-direction: column.

CSS 
/* flex-basis — taille initiale */
.basis-200 { flex-basis: 200px; }  /* commence à 200px avant grow/shrink */
.basis-auto { flex-basis: auto; }  /* selon le contenu (défaut) */
.basis-30  { flex-basis: 30%; }   /* 30% du conteneur */

/* ═══ SHORTHAND flex: grow shrink basis ═══ */
.flex-1   { flex: 1; }          /* flex: 1 1 0% — s'agrandit, peut rétrécir, taille 0 */
.flex-auto { flex: auto; }       /* flex: 1 1 auto — comme flex:1 mais taille selon contenu */
.flex-none { flex: none; }       /* flex: 0 0 auto — taille fixe, ne change pas */

/* Usage courant pour une sidebar + contenu principal */
.sidebar     { flex: 0 0 250px; }  /* sidebar fixe à 250px, ne s'agrandit ni rétrécit */
.main-content { flex: 1; }         /* prend tout l'espace restant */
RENDU
100px fixe
flex:1 (reste)
80px
💡
Les valeurs du shorthand flex à connaître

flex: 1 → flex-grow: 1; flex-shrink: 1; flex-basis: 0% = l'élément occupe tout l'espace disponible, partage également entre les flex:1. flex: auto → 1 1 auto = comme flex:1 mais la taille de base est celle du contenu. flex: none → 0 0 auto = taille fixe, ne s'adapte pas. Utilisez flex: 1 pour les mises en page responsive.

08

order & align-self

order modifie l'ordre d'affichage des éléments flexibles sans changer l'ordre dans le HTML. align-self permet à un élément individuel de se positionner différemment des autres sur l'axe secondaire.

CSS 
/* ORDER — réordonne visuellement (pas dans le DOM) */
.item-1 { order: 3; }  /* apparaît en 3e position */
.item-2 { order: 1; }  /* apparaît en 1er */
.item-3 { order: 2; }  /* apparaît en 2e */
/* Valeur par défaut : order: 0 */

/* ALIGN-SELF — surcharge align-items pour un seul élément */
.self-start   { align-self: flex-start; }  /* en haut */
.self-end     { align-self: flex-end; }    /* en bas */
.self-center  { align-self: center; }      /* centré */
.self-stretch { align-self: stretch; }     /* étiré */
.self-baseline{ align-self: baseline; }    /* ligne de base */
RENDU
1 (order:3)
2 align-self:start
3 (order:2)
4 align-self:end
⚠️
order et l'accessibilité

order modifie l'ordre visuel mais pas l'ordre dans le DOM. Les lecteurs d'écran et la navigation au clavier (Tab) suivent l'ordre du HTML, pas l'ordre CSS. N'utilisez jamais order pour changer l'ordre logique du contenu — uniquement pour du réordonnancement visuel (ex : déplacer un élément spécifique en dernier sur mobile).

09

Exercices pratiques

EX 01 🧩 Mise en page avec Flexbox
  1. Créez exo-flex-layout.html avec un conteneur flex horizontal de 4 cartes.
  2. Chaque carte a 25% de largeur initiale (flex: 1 ou flex-basis: 25%).
  3. Ajoutez flex-wrap: wrap pour que les cartes passent à la ligne sur écran étroit.
  4. Dans chaque carte : image (placeholder), titre, description, bouton.
  5. Utilisez gap: 20px pour espacer les cartes.
  6. Centrez le titre des cartes avec text-align: center.
EX 02 📐 Layout complexe avec Flexbox
  1. Créez exo-flex-complex.html avec : header, nav horizontal, contenu principal avec sidebar, footer.
  2. Le header utilise justify-content: space-between (logo à gauche, menu à droite).
  3. Le nav utilise display: flex avec gap pour les liens.
  4. La section principale a display: flex avec sidebar flex: 0 0 250px et contenu flex: 1.
  5. Dans la sidebar, utilisez flex-direction: column et gap pour les liens.
  6. Utilisez align-self sur un élément pour le placer en bas du footer.
🔍
Exercice bonus — Centrage parfait

Créez un carré de 300×300px avec un fond coloré. À l'intérieur, placez un texte de 3 lignes parfaitement centré horizontalement ET verticalement en utilisant display: flex; justify-content: center; align-items: center;. Ajoutez un sous-texte avec text-align: center. C'est la méthode la plus simple et la plus fiable pour centrer n'importe quoi.

Bilan & mini-glossaire

Ce que vous maîtrisez

  • display: flex sur le conteneur et le comportement des enfants flexibles
  • flex-direction : row (défaut), row-reverse, column, column-reverse
  • flex-wrap : nowrap, wrap, wrap-reverse et gap pour l'espacement
  • justify-content : flex-start, flex-end, center, space-between, space-around, space-evenly
  • align-items : stretch, flex-start, flex-end, center, baseline
  • flex-grow, flex-shrink, flex-basis et le shorthand flex
  • order pour le réordonnancement visuel
  • align-self pour le positionnement individuel sur l'axe secondaire

Mini-glossaire

Flexbox
Modèle de mise en page unidimensionnel (ligne ou colonne). Les enfants s'adaptent automatiquement.
Axe principal
Axe défini par flex-direction. Pour row : horizontal. Pour column : vertical.
Axe secondaire
Axe perpendiculaire à l'axe principal. Pour row : vertical. Pour column : horizontal.
justify-content
Alignement sur l'axe principal. Distribue l'espace entre les éléments.
align-items
Alignement sur l'axe secondaire. Appliqué à tous les enfants du conteneur.
flex-grow
Capacité d'un élément à s'agrandir pour occuper l'espace libre. Valeur par défaut : 0.
flex-shrink
Capacité d'un élément à rétrécir si l'espace manque. Valeur par défaut : 1.
flex-basis
Taille initiale d'un élément avant distribution de l'espace. Alternative à width/height.
flex: 1
Shorthand pour flex-grow: 1; flex-shrink: 1; flex-basis: 0%. Occupe l'espace disponible.
order
Réordonnancement visuel (pas dans le DOM). Valeur par défaut : 0.
align-self
Surcharge align-items pour un seul enfant. Valeurs : start, end, center, stretch.
